## 凝聚态物理学

凝聚态物理学的定义：从微观角度出发，研究相互作用多粒子系统组成的凝聚态物质的结构和动力学过程，及其与宏观物理过程之间关系的一门科学。

凝聚态物理的哲学是呈展论 (emergent)，其与高能物理和场论研究所秉持的还原论 (reductionism) 截然不同。

## 凝聚态物理学之父

Anderson是凝聚态物理学之父。

## 凝聚态理论

凝聚态理论是研究凝聚态物质时所发展的理论和方法，其基本观念包括：

对称性破缺 (Symmetry Breaking)：物质的相与对称性相联系，相变伴随对称性的变化。

准粒子 (Quasi-particle)：系统的低能激发可看成一些弱相互作用准粒子的集合。

拓扑序 (Topological Order)：物态在参数空间可拥有非平凡的拓扑结构。

# 2 二次量子化

## 一次量子化与二次量子化

在经典力学中，单粒子哈密顿量是正则坐标和正则动量的函数

$$
\begin{equation}
H=H(x,p)=\frac{p^2 }{2m } + V(x),
\end{equation}
$$

体系的动力学由如下的哈密顿方程给出：

$$
\begin{equation}
\dot{x} = \frac{\partial H }{\partial p } ,\quad
\dot{p} = -\frac{\partial H }{\partial x } 
\end{equation}
$$

一次量子化形式把坐标和动量换成算符 $x\to \hat{x},p\to \hat{p} $，并赋予正则对易关系

$$
\begin{equation}
\left[\hat{x} , \hat{p} \right] = \mathrm{i}\hbar
\end{equation}
$$

相应哈密顿量替换成哈密顿量算符

$$
\begin{equation}
H\to \hat{H} = \frac{\hat{p}^2 }{2m } + V(\hat{x})
\end{equation}
$$

粒子的运动状态由Hilbert空间中的态矢 $\Ket{\psi(t) } $ 描述，体系的动力学由薛定谔方程给出：

$$
\begin{equation}
\mathrm{i}\hbar\frac{\partial }{\partial t } \Ket{\psi(t) }
=\hat{H} \Ket{\psi(t) }
\end{equation}
$$

在坐标表象下即

$$
\begin{equation}
\mathrm{i}\hbar\frac{\partial }{\partial t } \psi(x,t)
=\Braket{x | \hat{H} | \psi(t) }
\end{equation}
$$

在二次量子化形式中，我们把波函数替换成场算符

$$
\begin{equation}
\psi\to \hat{\psi},\quad \psi^*\to \hat{\psi}^\dag
\end{equation}
$$

并赋予对易关系

$$
\begin{equation}
\left[\hat{\psi}(x) , \hat{\psi}^\dag(y) \right]_{\pm}
=\delta(x-y)
\end{equation}
$$

$+ $ 号代表反对易关系，对应费米子体系；$- $ 号代表对易关系，对应玻色子体系。

## 一次量子化形式中的多体波函数

我们知道，$\Ket{\bm{r}_1 }_1 \otimes \cdots\otimes \Ket{\bm{r}_N }_N $ 表示 $1 $ 号粒子处于 $\bm{r}_1,\cdots, $ $N $ 号粒子处于 $\bm{r}_N $ 的态。然而全同粒子不可区分，因此我们需要一个不依赖编号的态。

可以构造对称化且归一的态：

$$
\begin{equation}
\Ket{\bm{r}_1,\cdots,\bm{r}_N }_S
\equiv \sqrt{\frac{\prod }{ } } \sum_{\sigma\in S_N} \Ket{\bm{r}_{\sigma_1} }_1 \otimes \cdots\otimes \Ket{\bm{r}_{\sigma_N} }_N
\end{equation}
$$

也可以构造反对称化的态：

表示

$N $ 全同粒子体系波函数

$$
\begin{equation}
\psi\left(\bm{r}_1,\cdots,\bm{r}_N \right)
\equiv \Braket{\bm{r}_1,\cdots,\bm{r}_N | \psi }
\end{equation}
$$

假设有一组完备的单粒子波函数 $\left\{\phi_\alpha \right\} $，则可以构造出对称的全同粒子波函数（描述全同玻色子体系）：

$$
\begin{equation}
\psi_S(\bm{r}_1,\cdots,\bm{r}_N)
=\frac{1 }{\sqrt{N!} } \sum_{\sigma\in S_N} \phi_{\sigma_1}(\bm{r}_1) \cdots \phi_{\sigma_N}\phi(\bm{r}_N)
\end{equation}
$$

也可以构造出反对称的全同粒子波函数（描述全同费米子体系）：

$$
\begin{equation}
\psi_A(\bm{r}_1,\cdots,\bm{r}_N)
=\frac{1 }{\sqrt{N!} } \sum_{\sigma\in S_N} \mathrm{sign}(\sigma) \phi_{\sigma_1}(\bm{r}_1) \cdots \phi_{\sigma_N}\phi(\bm{r}_N)
\end{equation}
$$

对 $N $ 全同费米子体系，可以用多体Slater行列式来构造反对称波函数：

$$
\begin{equation}
\psi_A(\left\{\bm{r}_i \right\})
\equiv \frac{1 }{\sqrt{N!} } \mathrm{det}
\begin{pmatrix}
\phi_1(\bm{r}_1) &\phi_1(\bm{r}_2) &\cdots &\phi_1(\bm{r}_N) \\
\phi_2(\bm{r}_1) &\phi_2(\bm{r}_2) &\cdots &\phi_2(\bm{r}_N) \\
\vdots &\vdots &\ddots &\vdots \\
\phi_N(\bm{r}_1) &\phi_N(\bm{r}_2) &\cdots &\phi_N(\bm{r}_N) \\
\end{pmatrix}
\end{equation}
$$

### 谐振子的升降算符表示

谐振子哈密顿量

$$
\begin{equation}
H
=\frac{p^2 }{2m } + \frac{1 }{2 } m\omega^2 x^2
\end{equation}
$$

利用坐标算符和动量算符可以构造升降算符。反过来，可以用升降算符表示坐标算符和动量算符

$$
\begin{equation}
x = \sqrt{\frac{1 }{2m\omega } } \left(a + a^\dag \right),\quad
p = \mathrm{i}\sqrt{\frac{m\omega }{2 } } \left(-a + a^\dag \right)
\end{equation}
$$

哈密顿量可用升降算符表达为

$$
\begin{equation}
H
=\omega \left(a^\dag a + \frac{1 }{2 }  \right)
=\omega\left(\hat{n} +\frac{1 }{2 }  \right)
\end{equation}
$$

其中 $\hat{n}\equiv a^\dag a $ 是占据数算符。

占据数算符的本征解为：

$$
\begin{equation}
\hat{n} \Ket{n } = n\Ket{n },\quad n=0,1,2,\cdots,\quad
\Ket{n } = \frac{1 }{\sqrt{n!} } \left(a^\dag \right)^n \Ket{0 }
\end{equation}
$$

$$
\begin{equation}
a^\dag \Ket{n } = \sqrt{n+1} \Ket{n+1 },\quad
a \Ket{n } = \sqrt{n} \Ket{n-1 }
\end{equation}
$$

$$
\begin{equation}
\Braket{n | \hat{x} | n } = \Braket{n | \hat{p} | n } = 0
\end{equation}
$$

#### 玻色子

玻色子产生湮灭算符满足对易关系

$$
\begin{equation}
\left[a_i , a_j^\dag \right] = \delta_{ij},\quad
\left[a_i , a_j \right] = \left[a_i^\dag , a_j^\dag \right] = 0
\end{equation}
$$

用 $\Ket{n_1,n_2,\cdots,n_N } $ 表示有 $n_j $ 个粒子占据 $j $ 状态的态（由于是玻色子，$n_j $ 可以大于 $1 $）。

$$
\begin{equation}
\Ket{n_1,\cdots,n_j,\cdots }
\equiv \prod_i \frac{\left(a_i^\dag \right)^{n_i} }{\sqrt{n_i!} } \Ket{0 }
\end{equation}
$$

$$
\begin{equation}
a_j^\dag \Ket{n_1,\cdots,n_j,\cdots }
=\sqrt{n_j + 1} \Ket{n_1,\cdots,n_j+1,\cdots }
\end{equation}
$$

$$
\begin{equation}
a_j \Ket{n_1,\cdots,n_j,\cdots }
=\sqrt{n_j} \Ket{n_1,\cdots,n_j-1,\cdots }
\end{equation}
$$

$$
\begin{equation}
\hat{n}_j \Ket{n_1,\cdots,n_j,\cdots }
=n_j \Ket{n_1,\cdots,n_j,\cdots }
\end{equation}
$$

$$
\begin{equation}
a_j \Ket{0,0,\cdots,0 } = 0
\end{equation}
$$

### 多体哈密顿量的二次量子化形式

单体算符 $\displaystyle{\mathcal{O}=\sum_j o_j }$ 的二次量子化形式：

$$
\begin{equation}
\mathcal{O}
=\sum_{\alpha,\beta} \Braket{\alpha | o | \beta } a_\alpha^\dag a_\beta
\end{equation}
$$

其中 $\left\{\Ket{\alpha } \right\},\left\{\Ket{\beta } \right\} $ 单粒子完备基。

两体算符 $\displaystyle{V = \sum_{j\ne m} v_{jm} }$  的二次量子化形式：

$$
\begin{equation}
V
=\sum_{\alpha\beta\delta\gamma} \Braket{\alpha\beta | v | \delta \gamma } a_\alpha^\dag a_\beta^\dag a_\lambda a_\delta
\end{equation}
$$

其中 $\Ket{\delta \gamma }\equiv \Ket{\delta }\otimes \Ket{\gamma } $

考虑一个一般的包含单体和两体相互作用的多粒子习题哈密顿量

$$
\begin{equation}
H
=\sum_j \left[\frac{\hat{p}_j^2 }{2m } + U\left(\hat{x}_j \right) \right] + \frac{1 }{2 } \sum_{j\ne m} V\left(\hat{x}_j,\hat{x}_m \right)
\end{equation}
$$

现在想求在坐标表象下，该哈密顿量的二次量子化形式。

先看单体动能项的二次量子化形式，利用动量算符在坐标表象下对态的作用

$$
\begin{equation}
\Braket{x | \hat{p} | \Psi }
=-\mathrm{i}\hbar \nabla_x \Ket{\Psi }
=-\mathrm{i}\nabla_x \Ket{\Psi }
\end{equation}
$$

以及delta函数的高阶导数的筛选性质

$$
\begin{equation}
\int_{-\infty}^{+\infty} \delta^{(n)}(x-x') f(x) \mathrm{d}x
=(-1)^n f^{(n)}(x')
\end{equation}
$$

可得

$$
\begin{equation}
\begin{aligned}
\sum_j \frac{\hat{p}_j^2 }{2m } 
&=\int \mathrm{d}^dx \int\mathrm{d}^dy \Braket{x | \frac{\hat{p}^2 }{2m } | y } \psi^\dag(x) \psi(y) \\
&=\int \mathrm{d}^dx \int\mathrm{d}^dy \left(\frac{-\nabla_x^2 }{2m } \Braket{x | y } \right) \psi^\dag(x) \psi(y) \\
&=\int \mathrm{d}^dx \int\mathrm{d}^dy \left(\frac{-\nabla_x^2 }{2m } \delta(x-y) \right) \psi^\dag(x) \psi(y) \\
&=\int \mathrm{d}^dx \int\mathrm{d}^dy \left(\frac{-\nabla_x^2 }{2m } \delta(y-x) \right) \psi^\dag(x) \psi(y) \\
&=\int \mathrm{d}^dx \int\mathrm{d}^dy \left(\frac{-\nabla_y^2 }{2m } \delta(y-x) \right) \psi^\dag(x) \psi(y) \\
&=\int \mathrm{d}^dx \psi^\dag(x) \int\mathrm{d}^dy \left(\frac{-\nabla_y^2 }{2m } \delta(y-x) \right) \psi(y) \\
&=\int \mathrm{d}^dx \psi^\dag(x) \frac{-\nabla_x^2 }{2m } \psi(x)
\end{aligned}
\end{equation}
$$

再看单体势能项

$$
\begin{equation}
\begin{aligned}
\sum_j U\left(\hat{x}_j \right)
&=\int \mathrm{d}^dx \int\mathrm{d}^dy \Braket{x | U\left(\hat{x} \right) | y } \psi^\dag(x) \psi(y) \\
&=\int \mathrm{d}^dx \int\mathrm{d}^dy U\left(y \right) \Braket{x | y } \psi^\dag(x) \psi(y) \\
&=\int \mathrm{d}^dx \int\mathrm{d}^dy U\left(y \right) \delta(x-y) \psi^\dag(x) \psi(y) \\
&=\int \mathrm{d}^dx U\left(x \right) \psi^\dag(x) \psi(x) \\
&=\int \mathrm{d}^dx \psi^\dag(x) U\left(x \right) \psi(x) \\
\end{aligned}
\end{equation}
$$

再看两体势能项

$$
\begin{equation}
\begin{aligned}
\sum_{j\ne m} V\left(\hat{x}_j,\hat{x}_m \right)
&=\int \mathrm{d}^dx \mathrm{d}^dy\mathrm{d}^dx'\mathrm{d}^dy'\Braket{x,y | V\left(\hat{x},\hat{y} \right) | x',y' } \psi^\dag(x) \psi^\dag(y) \psi(y') \psi(x') \\
&=\int \mathrm{d}^dx \mathrm{d}^dy\mathrm{d}^dx'\mathrm{d}^dy' V\left(\hat{x},\hat{y} \right) \Braket{x,y | x',y' } \psi^\dag(x) \psi^\dag(y) \psi(y') \psi(x') \\
&=\int \mathrm{d}^dx \mathrm{d}^dy\mathrm{d}^dx'\mathrm{d}^dy' V\left(x',y' \right) \delta(x-x') \delta(y-y') \psi^\dag(x) \psi^\dag(y) \psi(y') \psi(x') \\
&=\int \mathrm{d}^dx \int \mathrm{d}^dy \psi^\dag(x) \psi^\dag(y) V(x,y) \psi(y) \psi(x) \\
\end{aligned}
\end{equation}
$$

综上，坐标表象二次量子化哈密顿量为

$$
\begin{equation}
\begin{aligned}
H
&=\sum_j \left[\frac{\hat{p}_j^2 }{2m } + U\left(\hat{x}_j \right) \right] + \frac{1 }{2 } \sum_{j\ne m} V\left(\hat{x}_j,\hat{x}_m \right) \\
&=\int \mathrm{d}^dx \psi^\dag(x) \left[\frac{-\nabla_x^2 }{2m } + U(x) \right] \psi(x) + \frac{1 }{2 } \int \mathrm{d}^dx \int \mathrm{d}^dy \psi^\dag(x) \psi^\dag(y) V(x,y) \psi(y) \psi(x) 
\end{aligned}
\end{equation}
$$

如果考虑电子系统，还要考虑自旋，此时

$$
\begin{equation}
\begin{aligned}
H
=\sum_{\sigma\in \left\{\uparrow,\downarrow \right\}} \int \mathrm{d}^dx \psi_\sigma^\dag(x) \left[\frac{-\nabla_x^2 }{2m } + U(x) \right] \psi_\sigma(x) + \frac{1 }{2 } \sum_{\sigma,\sigma'\in \left\{\uparrow,\downarrow \right\}} \int \mathrm{d}^dx \int \mathrm{d}^dy \psi_\sigma^\dag(x) \psi_{\sigma'}^\dag(y) V(x,y) \psi_{\sigma'}(y) \psi_{\sigma}(x) 
\end{aligned}
\end{equation}
$$

这里场算符满足如下的反对易关系

$$
\begin{equation}
\left\{\psi_\sigma(x) , \psi_{\sigma'}^\dag(y) \right\}
=\delta_{\sigma,\sigma'} \delta(x-y),\quad
\left\{\psi_\sigma(x) , \psi_{\sigma'}(y) \right\}
=\left\{\psi_\sigma^\dag(x) , \psi_{\sigma'}^\dag(y) \right\}
=0
\end{equation}
$$

### 微扰理论计算

当没有相互作用时，体系基态是费米球分布。

$$
\begin{equation}
\Ket{F }
=\prod_{k<k_F} c_{k\uparrow}^\dag c_{k\downarrow}^\dag \Ket{0 }
=\prod_{k\sigma} \Ket{n_{k\sigma} },\quad
n_{k\sigma} = \theta(k_F-k)
\end{equation}
$$

其中 $n_{k\sigma} $ 是动量为 $k $ 且自旋为 $\sigma $ 的电子的占据数。可以看出，无相互作用自由电子气体的占据数不依赖于自旋。

费米动量由基态时的总电子数决定

$$
\begin{equation}
\begin{aligned}
N
&=\Braket{F | \sum_{k,\sigma} c_{k\sigma}^\dag c_{k\sigma} | F }
=\sum_{k,\sigma} n_{k\sigma}
=\sum_{k,\sigma} \theta(k_F-k) \\
&=\sum_\sigma \int \frac{\mathrm{d}^3 k }{\left(2\pi \right)^3/V } \theta(k_F - k)
=\sum_\sigma \int_{k=0}^{k=k_F} \frac{4\pi k^2 \mathrm{d} k }{\left(2\pi \right)^3/V }
=\frac{V }{3\pi^2 } k_F^3
\end{aligned}
\end{equation}
$$

无相互作用自由电子气体基态能量：

$$
\begin{equation}
\begin{aligned}
E_g^0
&=\Braket{F | \sum_{k,\sigma} \frac{k^2 }{2m } c_{k\sigma}^\dag c_{k\sigma} | F }
=\sum_{k,\sigma} \frac{k^2 }{2m } n_{k\sigma}
=\sum_{k,\sigma} \frac{k^2 }{2m } \theta(k_F-k) \\
&=\sum_\sigma \int \frac{\mathrm{d}^3k }{\left(2\pi \right)^3/V } \frac{k^2 }{2m } \theta(k_F-k)
=\sum_\sigma \int_{k=0}^{k=k_F} \frac{4\pi k^2\mathrm{d}k }{\left(2\pi \right)^3/V } \frac{k^2 }{2m } \\
&=\frac{V }{2m\pi^2 } \frac{k_F^5 }{5 } 
\end{aligned}
\end{equation}
$$

加微扰后，体系基态能量精确到一阶修正：

$$
\begin{equation}
E_g
=E_g^0 + \Braket{F | H_{\mathrm{int}} | F }
\end{equation}
$$

基态能量的一阶修正：

$$
\begin{equation}
\begin{aligned}
\Braket{F | H_{\mathrm{int}} | F }
&=\frac{1 }{2 V} \sum_{k,k',q\ne 0,\sigma,\sigma'} \frac{4\pi e^2 }{q^2 } \Braket{F | c_{k+q,\sigma}^\dag c_{k'-q,\sigma'}^\dag c_{k',\sigma'} c_{k,\sigma} | F }
\end{aligned}
\end{equation}
$$

=-\Braket{\Psi | c_i^\dag c_l | \Psi } \Braket{\Psi | c_j^\dag c_m | \Psi } + \Braket{\Psi | c_i^\dag c_m | \Psi } \Braket{\Psi | c_j^\dag c_l | \Psi }
\end{equation}

\end{equation}

### Hartree-Fock近似（平均场近似）

$$
\begin{equation}
A B 
\approx \Braket{A } B + A \Braket{B } - \Braket{A }\Braket{B }
\end{equation}
$$

这里 $\Braket{ } $ 代表系综平均或基态期望。

对于Coulomb电子气，对相互作用项中的四费米子项用H-F近似：

$$
\begin{equation}
H_{\mathrm{int}}
=\frac{1 }{2 V} \sum_{k,k',q\ne 0,\sigma,\sigma'} \frac{4\pi e^2 }{q^2 }  c_{k+q,\sigma}^\dag c_{k'-q,\sigma'}^\dag c_{k',\sigma'} c_{k,\sigma}
\end{equation}
$$

$$
\begin{equation}
\begin{aligned}
&c_{k+q,\sigma}^\dag c_{k'-q,\sigma'}^\dag c_{k',\sigma'} c_{k,\sigma} \\
=&-c_{k+q,\sigma}^\dag c_{k',\sigma'} c_{k'-q,\sigma'}^\dag c_{k,\sigma} \\
\approx& -\Braket{c_{k+q,\sigma}^\dag c_{k',\sigma'} } c_{k'-q,\sigma'}^\dag c_{k,\sigma} - c_{k+q,\sigma}^\dag c_{k',\sigma'} \Braket{c_{k'-q,\sigma'}^\dag c_{k,\sigma} } + \Braket{c_{k+q,\sigma}^\dag c_{k',\sigma'} } \Braket{c_{k'-q,\sigma'}^\dag c_{k,\sigma} } \\
=&-\delta_{k',k+q} \delta_{\sigma,\sigma'} \Braket{c_{k',\sigma'}^\dag c_{k',\sigma'} } c_{k'-q,\sigma'}^\dag c_{k,\sigma} - \delta_{k,k'-q} \delta_{\sigma,\sigma'} \Braket{c_{k,\sigma}^\dag c_{k,\sigma} } c_{k+q,\sigma}^\dag c_{k',\sigma'} + \delta_{k',k+q} \delta_{\sigma,\sigma'} \delta_{k,k'-q} \Braket{c_{k',\sigma'}^\dag c_{k',\sigma'} } \Braket{c_{k,\sigma}^\dag c_{k,\sigma} } \\
=&-\delta_{k',k+q} \delta_{\sigma,\sigma'} \Braket{c_{k',\sigma'}^\dag c_{k',\sigma'} } c_{k,\sigma}^\dag c_{k,\sigma} - \delta_{k,k'-q} \delta_{\sigma,\sigma'} \Braket{c_{k,\sigma}^\dag c_{k,\sigma} } c_{k',\sigma'}^\dag c_{k',\sigma'} + \delta_{k',k+q} \delta_{\sigma,\sigma'} \delta_{k,k'-q} \Braket{c_{k',\sigma'}^\dag c_{k',\sigma'} } \Braket{c_{k,\sigma}^\dag c_{k,\sigma} } \\
\end{aligned}
\end{equation}
$$

忽略最后的常数项就有

$$
\begin{equation}
\begin{aligned}
c_{k+q,\sigma}^\dag c_{k'-q,\sigma'}^\dag c_{k',\sigma'} c_{k,\sigma}
&\approx -\delta_{k',k+q} \delta_{\sigma,\sigma'} \Braket{c_{k',\sigma'}^\dag c_{k',\sigma'} } c_{k,\sigma}^\dag c_{k,\sigma} - \delta_{k,k'-q} \delta_{\sigma,\sigma'} \Braket{c_{k,\sigma}^\dag c_{k,\sigma} } c_{k',\sigma'}^\dag c_{k',\sigma'} \\
&=-\delta_{k',k+q} \delta_{\sigma,\sigma'} n_{k',\sigma'} c_{k,\sigma}^\dag c_{k,\sigma} - \delta_{k,k'-q} \delta_{\sigma,\sigma'} n_{k,\sigma} c_{k',\sigma'}^\dag c_{k',\sigma'} \\
&=-\delta_{k',k+q} \delta_{\sigma,\sigma'} \theta(k_F-k') c_{k,\sigma}^\dag c_{k,\sigma} - \delta_{k,k'-q} \delta_{\sigma,\sigma'} \theta(k_F-k) c_{k',\sigma'}^\dag c_{k',\sigma'} \\
\end{aligned}
\end{equation}
$$

则

$$
\begin{equation}
\begin{aligned}
H_{\mathrm{int}}
&=\frac{1 }{2 V} \sum_{k,k',q\ne 0,\sigma,\sigma'} \frac{4\pi e^2 }{q^2 }  c_{k+q,\sigma}^\dag c_{k'-q,\sigma'}^\dag c_{k',\sigma'} c_{k,\sigma} \\
&\approx \frac{1 }{2 V} \sum_{k,k',q\ne 0,\sigma,\sigma'} \frac{4\pi e^2 }{q^2 } \left[-\delta_{k',k+q} \delta_{\sigma,\sigma'} \theta(k_F-k') c_{k,\sigma}^\dag c_{k,\sigma} - \delta_{k,k'-q} \delta_{\sigma,\sigma'} \theta(k_F-k) c_{k',\sigma'}^\dag c_{k',\sigma'} \right] \\
&=-\frac{1 }{2V } \sum_{k,k',\sigma} \frac{4\pi e^2 }{\left|k'-k \right|^2 } \left[\theta(k_F-k') c_{k,\sigma}^\dag c_{k,\sigma} + \theta(k_F-k) c_{k',\sigma}^\dag c_{k',\sigma} \right] \\
&=-\frac{1 }{2V } \sum_{k,k',\sigma} \frac{4\pi e^2 }{\left|k'-k \right|^2 } \left[\theta(k_F-k') c_{k,\sigma}^\dag c_{k,\sigma} + \theta(k_F-k') c_{k,\sigma}^\dag c_{k,\sigma} \right] \\
&=-\frac{1 }{V } \sum_{k,k',\sigma} \frac{4\pi e^2 }{\left|k'-k \right|^2 } \theta(k_F-k') c_{k,\sigma}^\dag c_{k,\sigma} \\
\end{aligned}
\end{equation}
$$

等效单电子能量：

$$
\begin{equation}
\varepsilon_k
=\frac{k^2 }{2m } - \frac{1 }{V } \sum_{k'} \frac{4\pi e^2 }{\left|k'-k \right|^2 } \theta(k_F-k')
\end{equation}
$$

交换项或Fock项：

$$
\begin{equation}
\begin{aligned}
-\varepsilon^{\mathrm{ex}}_k
=\frac{e^2 k_F }{\pi } \left(1 + \frac{k_F^2-k^2 }{2k_F k } \ln\left|\frac{k+k_F }{k-k_F }  \right| \right)
\end{aligned}
\end{equation}
$$

H-F近似下单电子能量：

$$
\begin{equation}

\end{equation}

=&-\delta_{k+q,k'} \delta_{\sigma,\sigma'} \Braket{c_{k',\sigma'}^\dagger c_{k'\sigma'} } \delta_{k'-q,k} \delta_{\sigma,\sigma'} \Braket{c_{k,\sigma}^\dagger c_{k\sigma} } + \delta_{k+q,k} \Braket{c_{k,\sigma}^\dagger c_{k\sigma}} \delta_{k'-q,k'} \Braket{c_{k',\sigma'}^\dagger c_{k'\sigma'}} \\
=&-\delta_{q,k'-k} \delta_{\sigma,\sigma'} \Braket{c_{k',\sigma'}^\dagger c_{k'\sigma'} } \Braket{c_{k,\sigma}^\dagger c_{k\sigma} } + \delta_{q,0} \Braket{c_{k,\sigma}^\dagger c_{k\sigma}} \Braket{c_{k',\sigma'}^\dagger c_{k'\sigma'}} \\
=&-\delta_{q,k'-k} \delta_{\sigma,\sigma'} \theta(k_F-k') \theta(k_F-k) + \delta_{q,0} \theta(k_F-k) \theta(k_F-k') \\
\end{aligned}
\end{equation}
$$

基态能量一阶修正为：

$$
\begin{equation}
\begin{aligned}
E_g^1
&=\Braket{F | H_{\mathrm{int}} | F } \\
&=\frac{\lambda}{2V} \sum_{k,k',q,\sigma,\sigma'} \Braket{c_{k+q,\sigma}^\dagger c_{k'-q,\sigma'}^\dagger c_{k'\sigma'} c_{k\sigma} } \\
&=\frac{\lambda}{2V} \sum_{k,k',q,\sigma,\sigma'} \left[-\delta_{q,k'-k} \delta_{\sigma,\sigma'} \theta(k_F-k') \theta(k_F-k) + \delta_{q,0} \theta(k_F-k) \theta(k_F-k') \right] \\
&=\frac{\lambda}{2V} \sum_{k,k'} \left[-2\theta(k_F-k') \theta(k_F-k) + 4\theta(k_F-k') \theta(k_F-k) \right] \\
&=\frac{\lambda }{V } \sum_{k,k'} \theta(k_F-k') \theta(k_F-k) \\
&=\frac{\lambda }{V } \left[\sum_k \theta(k_F-k) \right]^2 \\
&=\frac{\lambda }{V } \left[\int \frac{\mathrm{d}^3k }{\left(2\pi \right)^3/V } \theta(k_F-k)  \right]^2 \\
&=\frac{\lambda }{V } \left[\int_{k=0}^{k=k_F} \frac{4\pi k^2 \mathrm{d}k }{\left(2\pi \right)^3/V }  \right]^2 \\
&=\frac{\lambda }{V } \cdot \frac{\left(4\pi \right)^2 V^2 }{\left(2\pi \right)^6 } \left(\frac{k_F^3 }{3 }  \right)^2 \\
&=\lambda V \left( \frac{4\pi}{3} \right)^2 \left( \frac{k_F}{2\pi} \right)^6
\end{aligned}
\end{equation}
$$

一维最近邻耦合铁磁量子Heisenberg模型模型：

$$
\begin{equation}

\equiv S_j^x \pm \mathrm{i} S_j^y

改写哈密顿量

$$
\begin{equation}
H_{FM}
=-J\sum_j \left[\frac{1 }{2 } \left(S_j^+ S_{j+1}^- + S_j^- S_{j+1}^+ \right) + S_j^z S_{j+1}^z \right]

$$

其中 $R_i $ 是格点 $i $ 的位置。

## Landau平均场理论

假设自由能是序参量的函数。

有序态时序参量非零，无序态时序参量为零。

在相变点附近，序参量趋于零，可以把自由能密度按序参量 $\Psi $ 展开

$$
\begin{equation}
f_L\left[\Psi \right]
\equiv \frac{F_L\left[\Psi \right] }{V } 
=f_0 + \frac{r }{2 } \Psi^2 + \frac{u }{4 } \Psi^4 + \cdots
\end{equation}
$$

通常假设 $r $ 是温度的函数，$u>0 $ 并忽略高阶项。

$$
\begin{equation}
r = a(T-T_c),\quad a>0
\end{equation}
$$

$$
\begin{equation}
\frac{\partial f_L\left[\Psi \right] }{\partial \Psi } = 0 
\Longrightarrow \Psi \left(r + u \Psi^2 \right) = 0,\quad
\Psi = 0,\quad \Psi = \pm\sqrt{-\frac{r }{u } }
\end{equation}
$$

当 $T>T_c $，有 $r>0 $，物理上 $\Psi=0 $ 的零点对应系统稳定时的态，而我们能说 $\Psi=0 $ 对应无序态，因此温度大于临界温度时，体系处于无序态。此时自由能密度为

$$
\begin{equation}
f_L\left[\Psi=0 \right] = 0,\quad T>T_c
\end{equation}
$$

当 $T<T_c $，有 $r<0 $，此时 $\Psi=\pm\sqrt{-r/u} $ 这两个零点对应自由能密度的最小值，而我们说序参量非零对应有序态，因此温度小于临界温度时，系统进入有序态。此时自由能密度为

$$
\begin{equation}
f_L\left[\Psi\ne 0 \right]
=-\frac{r^2 }{4u },\quad T<T_c
\end{equation}
$$

### Landau平均场理论的可观测量

在Landau理论中，当 $T>T_c $ 时，$\Psi\equiv 0,f_L=0 $，因此热容 $C_V\equiv 0 .$

当 $T<T_c $ 时，自由能密度

$$
\begin{equation}
f_L
=-\frac{r^2 }{4u } 
=-\frac{\left(T-T_c \right)^2 }{4u } 
\end{equation}
$$

单位体积热容

$$
\begin{equation}
C_V
=-T\frac{\partial^2 f_L }{\partial T^2 } 
=\frac{T }{2u } 
\end{equation}
$$

在相变点，热容跳变

$$
\begin{equation}
\left. \Delta C_V \right|_{T_c} 
=\left. C_V \right|_{T\to T_c^-} - \left. C_V \right|_{T\to T_c^+}
=\frac{T_c }{2u } 
\end{equation}
$$

如果考虑外场下的物理量，比如磁场，那依赖于序参量的自由能密度写为

$$
\begin{equation}
f_L\left[\Psi \right]
=\frac{r }{2 } \Psi + \frac{u }{4 } \Psi^4 - h\Psi
\end{equation}
$$

$$
\begin{equation}

\Longrightarrow r \Psi + u \Psi^3 = h
\end{equation}
$$

由于只考虑相变点附近，因此 $\Psi^3 $ 是高阶小量，约去得到

$$
\begin{equation}
\Psi =\frac{h }{r } 
\end{equation}
$$

序参量的磁化率为

$$
\begin{equation}
\chi
\equiv \frac{\mathrm{d}\Psi }{\mathrm{d}h } 
=\frac{1 }{r } 
=\frac{1 }{T-T_c } 
\end{equation}
$$

在相变点 $T=T_c $ 有 $r=0 $，此时有

$$
\begin{equation}
\Psi = \left(\frac{h }{u }  \right)^{1/3}
\end{equation}

考虑费米面外两个电子，相互作用 $V(r_1-r_2) $，且与费米子球内电子无相互作用，哈密顿量可用质心坐标 $R=(r_1+r_2)/2 $ 和相对坐标 $r=r_1-r_2 $ 表达为

$$
\begin{equation}
H
=-\frac{1 }{4m } \frac{\partial^2 }{\partial R^2 } - 2 \frac{1 }{2 m} \frac{\partial^2 }{\partial r^2 } +V(r)
\end{equation}
$$

质心运动与相对运动分量，本征波函数的质心运动部分是平面波

$$
\begin{equation}
\Psi(R,r)
=\frac{1 }{\sqrt{V} } \mathrm{e}^{\mathrm{i}\vec{q}\cdot\vec{R}} \psi(r)
\end{equation}
$$

相对运动波函数满足本征方程

$$
\begin{equation}
\left[- 2 \frac{1 }{2 m} \frac{\partial^2 }{\partial r^2 } +V(r) \right] \psi(r)
=\left(E - \frac{q^2 }{4m }  \right) \psi(r)
\end{equation}
$$

对 $\psi(r) $ 和 $V(r) $ 进行傅里叶分解

$$
\begin{equation}
\psi(r)
=\frac{1 }{\sqrt{V} } \sum_k \mathrm{e}^{\mathrm{i}kr} g(k),\quad
V(r)
=\sum_{k'} \mathrm{e}^{\mathrm{i}k'r} V_{k'}
\end{equation}
$$

后得到

$$
\begin{equation}
\sum_k \left(2 \frac{k^2 }{2m } - E + \frac{q^2 }{4m }   \right) \mathrm{e}^{\mathrm{i}kr} g(k) + \sum_{k,k'} \mathrm{e}^{\mathrm{i}\left(k+k' \right)r} g(k) V_{k'}
=0
\end{equation}
$$

一番变形并引入相对费米能得单电子能量 $\varepsilon_k \equiv k^2/2m - E_F $ 后得到

$$
\begin{equation}
\left(2 \varepsilon_k + 2E_F - E + \frac{q^2 }{4m }  \right) g(k) + \sum_{\tilde{k}} V_{k-\tilde{k}} g\left(\tilde{k} \right)
=0
\end{equation}
$$

假定如下的相互作用势

$$
V_{k-\tilde{k}}
=\begin{cases}
-g / V&,0<\varepsilon_k,\varepsilon_{\tilde{k}}<\omega_c \\
0&,\mathrm{otherwise}
\end{cases}
$$

可得

$$
\begin{equation}
1
=\frac{g }{V } \sum_k \frac{1 }{2\varepsilon_k + 2E_F - E +q^2/4m } 
\end{equation}
$$

求和换成积分

$$
\begin{equation}
1
=g \int_0^{\omega_c} \mathrm{d}\varepsilon \frac{N(\varepsilon) }{2 \varepsilon + 2 E_F - E +q^2/4m } 
\end{equation}
$$

$N(\varepsilon)\approx N(0) $ 给出

$$
\begin{equation}
E
=2 E_F + \frac{q^2 }{4m } - \left(2 \omega_c + 2 E_F - E + \frac{q^2 }{4m }  \right) \mathrm{e}^{-\frac{2 }{g N(0) } }
\end{equation}
$$

弱耦合极限下有

$$
\begin{equation}
E
\approx 2 E_F + \frac{q^2 }{4m } - 2\omega_c \mathrm{e}^{-\frac{2 }{g N(0) } }
\end{equation}
$$

体系最小能量

$$
\begin{equation}
E_{\min}
=2 E_F - 2\omega_c \mathrm{e}^{-\frac{2 }{g N(0) } }
\end{equation}
$$

=\sum_{k,\sigma} \left(\varepsilon_k - \mu \right) c^\dag_{k\sigma} c_{k\sigma} + \sum_{k,k'} V_{k,k'} c^\dag_{k\uparrow} c^\dag_{-k\downarrow} c_{-k'\downarrow} c_{k'\uparrow}
\end{aligned}
\end{equation}
$$

假设配对 $V_{k,k'} $ 可写成

$$
\begin{equation}
\begin{aligned}
V_{k,k'}
=-\frac{g }{V } \gamma_k \gamma_{k'},\quad
\gamma_k\in \mathbb{R}
\end{aligned}
\end{equation}
$$

则哈密顿量化为

$$
\begin{equation}
\begin{aligned}
H
=\sum_{k,\sigma} \left(\varepsilon_k - \mu \right) c^\dag_{k\sigma} c_{k\sigma} - \frac{g }{V } \left(\sum_{k} \gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} \right) \left(\sum_{k'} \gamma_{k'} c_{-k'\downarrow} c_{k'\uparrow} \right)
\end{aligned}
\end{equation}
$$

考虑平均场近似：

$$
\begin{equation}
\begin{aligned}
&\left(\sum_{k} \gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} \right) \left(\sum_{k'} \gamma_{k'} c_{-k'\downarrow} c_{k'\uparrow} \right) \\
\approx&\left(\sum_{k} \gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} \right) \left(\sum_{k'} \gamma_{k'} \Braket{c_{-k'\downarrow} c_{k'\uparrow} } \right) + \left(\sum_{k} \gamma_k \Braket{c^\dag_{k\uparrow} c^\dag_{-k\downarrow} } \right) \left(\sum_{k'} \gamma_{k'} c_{-k'\downarrow} c_{k'\uparrow} \right) - \left(\sum_{k} \Braket{\gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} } \right) \left(\sum_{k'} \gamma_{k'} \Braket{c_{-k'\downarrow} c_{k'\uparrow} } \right) \\
\end{aligned}
\end{equation}
$$

定义超导能隙

$$
\begin{equation}
\begin{aligned}
\Delta
\equiv -\frac{g }{V } \sum_{k'} \gamma_{k'} \Braket{c_{-k'\downarrow} c_{k'\uparrow} },\quad
\Delta^*
=-\frac{g }{V } \sum_{k} \gamma_k \Braket{c^\dag_{k\uparrow} c^\dag_{-k\downarrow} }
\end{aligned}
\end{equation}
$$

则

$$
\begin{equation}
\begin{aligned}
&\left(\sum_{k} \gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} \right) \left(\sum_{k'} \gamma_{k'} c_{-k'\downarrow} c_{k'\uparrow} \right) \\
\approx&\left(\sum_{k} \gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} \right) \left(\sum_{k'} \gamma_{k'} \Braket{c_{-k'\downarrow} c_{k'\uparrow} } \right) + \left(\sum_{k} \gamma_k \Braket{c^\dag_{k\uparrow} c^\dag_{-k\downarrow} } \right) \left(\sum_{k'} \gamma_{k'} c_{-k'\downarrow} c_{k'\uparrow} \right) - \left(\sum_{k} \Braket{\gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} } \right) \left(\sum_{k'} \gamma_{k'} \Braket{c_{-k'\downarrow} c_{k'\uparrow} } \right) \\
=&\frac{-V\Delta }{g }  \left(\sum_{k} \gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} \right) + \frac{-V \Delta^* }{g }  \left(\sum_{k'} \gamma_{k'} c_{-k'\downarrow} c_{k'\uparrow} \right) - \frac{V^2 \left|\Delta \right|^2 }{g^2 }  \\
\end{aligned}
\end{equation}
$$

于是平均场哈密顿量为：

$$
\begin{equation}
\begin{aligned}
H
&=\sum_{k,\sigma} \left(\varepsilon_k - \mu \right) c^\dag_{k\sigma} c_{k\sigma} - \frac{g }{V } \left(\sum_{k} \gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} \right) \left(\sum_{k'} \gamma_{k'} c_{-k'\downarrow} c_{k'\uparrow} \right) \\
&\approx \sum_{k,\sigma} \left(\varepsilon_k - \mu \right) c^\dag_{k\sigma} c_{k\sigma} - \frac{g }{V } \left[\left(\sum_{k} \gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} \right) \left(\sum_{k'} \gamma_{k'} \Braket{c_{-k'\downarrow} c_{k'\uparrow} } \right) + \left(\sum_{k} \gamma_k \Braket{c^\dag_{k\uparrow} c^\dag_{-k\downarrow} } \right) \left(\sum_{k'} \gamma_{k'} c_{-k'\downarrow} c_{k'\uparrow} \right) - \left(\sum_{k} \Braket{\gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} } \right) \left(\sum_{k'} \gamma_{k'} \Braket{c_{-k'\downarrow} c_{k'\uparrow} } \right) \right] \\
&=\sum_{k,\sigma} \left(\varepsilon_k - \mu \right) c^\dag_{k\sigma} c_{k\sigma} - \frac{g }{V } \left[\frac{-V\Delta }{g }  \left(\sum_{k} \gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} \right) + \frac{-V \Delta^* }{g }  \left(\sum_{k'} \gamma_{k'} c_{-k'\downarrow} c_{k'\uparrow} \right) - \frac{V^2 \left|\Delta \right|^2 }{g^2 } \right] \\
&=\sum_{k,\sigma} \left(\varepsilon_k - \mu \right) c^\dag_{k\sigma} c_{k\sigma} + \sum_{k} \left(\Delta \gamma_k \right) c^\dag_{k\uparrow} c^\dag_{-k\downarrow} + \sum_{k'} \left(\Delta^* \gamma_{k'} \right) c_{-k'\downarrow} c_{k'\uparrow} + \frac{V }{g } \left|\Delta \right|^2  \\
\end{aligned}
\end{equation}
$$

定义超导配对能隙函数

$$
\begin{equation}
\begin{aligned}
\Delta_k
\equiv \Delta \gamma_k,\quad
\Delta_{k'}^*
=\Delta^* \gamma_{k'}
\end{aligned}
\end{equation}
$$

定义相对于化学势的单电子能量

$$
\begin{equation}
\begin{aligned}
\xi_k
\equiv \varepsilon_k -\mu
\end{aligned}
\end{equation}
$$

最终得到BCS平均场哈密顿量：

$$
\begin{equation}
\begin{aligned}
H
=\sum_{k,\sigma} \xi_k c^\dag_{k\sigma} c_{k\sigma} + \sum_{k} \Delta_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} + \sum_{k'} \Delta_k^* c_{-k'\downarrow} c_{k'\uparrow} + \frac{V }{g } \left|\Delta \right|^2
\end{aligned}
\end{equation}
$$

$$
\begin{equation}
\begin{aligned}

&=\sum_{k,\sigma} \xi_k \left(c^\dag_{k\uparrow} c_{k\uparrow} + c^\dag_{k\downarrow} c_{k\downarrow} \right) + \sum_{k} \Delta_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} + \sum_{k'} \Delta_k^* c_{-k'\downarrow} c_{k'\uparrow} + \frac{V }{g } \left|\Delta \right|^2 \\

\end{aligned}
\end{equation}
$$

#### Bogoliubov变换

$$
\begin{equation}
\begin{aligned}
H
&=\sum_k
\begin{pmatrix}
c_{k\uparrow}^\dag &c_{-k\downarrow}
\end{pmatrix}
\begin{pmatrix}
\xi_k &\Delta_k \\

\end{pmatrix}
\begin{pmatrix}
c_{k\uparrow} \\
c_{-k\downarrow}^\dag
\end{pmatrix}
+\frac{V }{g } \left|\Delta \right|^2 + \sum_k \xi_k \\
&\equiv \sum_k \psi_k^\dag h_k \psi_k + \frac{V }{g } \left|\Delta \right|^2 + \sum_k \xi_k \\
\end{aligned}
\end{equation}
$$

把 $h_k $ 对角化，得到

$$
\begin{equation}
h_k
=U_k D_k U_k^\dag
\end{equation}
$$

其中

$$
\begin{equation}
U_k
=\begin{pmatrix}
u_k &-v_k^* \\
v_k &u_k^*
\end{pmatrix},\quad
u_k = \sqrt{\frac{1 }{2 } \left(1+\frac{\xi_k }{E_k }  \right) },\quad
v_k = \frac{\Delta_k^* }{\left|\Delta_k \right| } \sqrt{\frac{1 }{2 } \left(1-\frac{\xi_k }{E_k }  \right)}
\end{equation}
$$

$$
\begin{equation}
D_k
=\mathrm{diag}(E_{k},-E_{k}),\quad
E_k = \sqrt{\xi_k^2 + \left|\Delta_k \right|^2}>0
\end{equation}
$$

$$
\begin{equation}
\begin{pmatrix}
\alpha_k \\
\beta_k^\dag
\end{pmatrix}

\begin{pmatrix}
c_{k\uparrow} \\
c_{-k\downarrow}^\dag
\end{pmatrix}
\end{equation}
$$

则

$$
\begin{equation}
H
=\sum_k E_k \left(\alpha_k^\dag \alpha_k + \beta_k^\dag \beta_k \right) + \frac{V }{g } \left|\Delta \right|^2 + \sum_k \left(\xi_k-E_k \right)
\end{equation}
$$

\end{aligned}
\end{equation}
$$
