## 凝聚态物理学

凝聚态物理学的定义：从微观角度出发，研究相互作用多粒子系统组成的凝聚态物质的结构和动力学过程，及其与宏观物理过程之间关系的一门科学。

凝聚态物理的哲学是呈展论 (emergent)，其与高能物理和场论研究所秉持的还原论 (reductionism) 截然不同。

## 凝聚态物理学之父

Anderson是凝聚态物理学之父。

## 凝聚态理论

凝聚态理论是研究凝聚态物质时所发展的理论和方法，其基本观念包括：

对称性破缺 (Symmetry Breaking)：物质的相与对称性相联系，相变伴随对称性的变化。

准粒子 (Quasi-particle)：物质的低能激发可看成一些弱相互作用准粒子的集合。

拓扑序 (Topological Order)：物态在参数空间可拥有非平凡的拓扑结构。

## 二次量子化

### 场算符

用 $\Ket{\mathrm{vac} } $ 表示真空态。

场算符（以坐标为参数）的定义如下：

$$
\hat{\psi}^\dag(x) \Ket{\mathrm{vac} }
=\Ket{x }
$$

其中，$\Ket{x } $ 是坐标算符 $\hat{x} $ 对应于本征值 $x $ 的本征态，满足：

$$
\hat{x} \Ket{x } = x \Ket{x }
$$

$\Ket{x } $ 代表一个处于位置 $x $ 的态，因此 $\hat{\psi}^\dag(x) $ 就是产生算符。

### 算符的二次量子化形式

单体算符 $\displaystyle{\hat{O}=\sum_{i=1}^{N} \hat{o}_i }$：

$$
\hat{O}
=\int \mathrm{d}\alpha \int \mathrm{d}\beta \Braket{\alpha | \hat{o} | \beta } \hat{a}^\dag(\alpha) \hat{a}(\beta)
$$

两体算符 $\displaystyle{\hat{V}=\frac{1 }{2 }\sum_{i=1}^{N} \sum_{j=1;j\ne i}^{N}\hat{v}_{ij} }$ 

$$
\hat{V}
=\int \mathrm{d}\alpha\mathrm{d}\beta\mathrm{d}\gamma\mathrm{d}\delta \Braket{\alpha \beta | \hat{v} | \gamma \delta } \hat{a}^\dag(\alpha) \hat{a}^\dag(\beta) \hat{a}(\delta) \hat{a}(\gamma)
$$

## 电子气模型

### 计算动量算符在坐标表象的矩阵元（表示）

现在想要计算：$\Braket{x|\hat{p}|x' } .$

对易关系

$$
\hat{x} \hat{p} - \hat{p} \hat{x} = \mathrm{i}\hbar
$$

$$
\Braket{x|\hat{x} \hat{p}|x' } - \Braket{x|\hat{p} \hat{x}|x' } = \mathrm{i}\hbar \Braket{x|x' }
$$

$$
\boxed{
\left(x-x' \right)\Braket{x|\hat{p}|x' } = \mathrm{i}\hbar \delta(x-x')
}
$$

#### $\delta $ 函数的导数的筛选性质

$\forall a>x' $ 有

$$
\begin{aligned}
\int_{x=x'-a}^{x=x'+a} f(x) \frac{\partial \delta(x-x') }{\partial x } \mathrm{d}x
&=f(x)\delta(x-x')\bigg|_{x=x'-a}^{x=x'+a} - \int_{x=x'-a}^{x=x'+a} f'(x) \delta(x-x')\mathrm{d}x \\
&=- \int_{x=x'-a}^{x=x'+a} f'(x) \delta(x-x')\mathrm{d}x
\end{aligned}
$$

对比可得：

$$
\boxed{
f(x) \frac{\partial \delta(x-x') }{\partial x }
=-f'(x) \delta(x-x')
}
$$

令 $f(x)=-\mathrm{i}\hbar(x-x'),f'(x)=-\mathrm{i}\hbar $ 就得到：

$$
\boxed{
(x-x')\left(-\mathrm{i}\hbar \frac{\partial \delta(x-x') }{\partial x } \right)
=\mathrm{i}\hbar\delta(x-x')
}
$$

上式与 $\left(x-x' \right)\Braket{x|\hat{p}|x' } = \mathrm{i}\hbar \delta(x-x') $ 对比可得：

$$
\Braket{x|\hat{p}|x' } = -\mathrm{i}\hbar \frac{\partial \delta(x-x') }{\partial x } 
$$

也即：

$$
\boxed{
\Braket{x|\hat{p}|x' }
=-\mathrm{i}\hbar \frac{\partial \delta(x-x') }{\partial x } 
=\mathrm{i}\hbar \frac{\partial \delta(x-x') }{\partial x' } 
}
$$

#### 证明 $\displaystyle{\Braket{x|\hat{p}|\psi}=-\mathrm{i}\hbar\frac{\partial }{\partial x } \Braket{x|\psi } }$ 

其中 $\Ket{\psi } $ 是任意态。

在 $\Braket{x|\hat{p}|\psi} $ 中插入完备性关系，并利用 $\displaystyle{\Braket{x|\hat{p}|x' }=-\mathrm{i}\hbar \frac{\partial \delta(x-x') }{\partial x } }$：

$$
\begin{aligned}
\Braket{x|\hat{p}|\psi}
&=\int \mathrm{d}x' \Braket{x|\hat{p}|x' } \Braket{x'|\psi } \\
&=\int \mathrm{d}x' \left[-\mathrm{i}\hbar \frac{\partial \delta(x-x') }{\partial x } \Braket{x'|\psi } \right] \\
&=-\mathrm{i}\hbar \frac{\partial }{\partial x } \int \Braket{x'|\psi } \delta(x-x') \mathrm{d}x' \\
&=-\mathrm{i}\hbar \frac{\partial }{\partial x } \Braket{x|\psi }  \\
\end{aligned}
$$

#### 计算 $\Braket{x|\hat{p}^2|\psi } $

$$
\begin{aligned}
\Braket{x|\hat{p}^2|\psi }
&=\Bra{x} \hat{p} \left(\hat{p} \Ket{\psi } \right) \\
&=-\mathrm{i}\hbar \frac{\partial }{\partial x } \Braket{x|\hat{p}|\psi } \\
&=-\mathrm{i}\hbar \frac{\partial }{\partial x } \left(-\mathrm{i}\hbar \frac{\partial }{\partial x } \Braket{x|\psi } \right) \\
&=-\hbar^2 \frac{\partial^2 }{\partial x^2 } \Braket{x|\psi }
\end{aligned}
$$

## BCS超导的平均场理论

### BCS平均场哈密顿量

BCS哈密顿量为

$$
\begin{equation}
\begin{aligned}
H
=\sum_{k,\sigma} \left(\varepsilon_k - \mu \right) c^\dag_{k\sigma} c_{k\sigma} + \sum_{k,k'} V_{k,k'} c^\dag_{k\uparrow} c^\dag_{-k\downarrow} c_{-k'\downarrow} c_{k'\uparrow}
\end{aligned}
\end{equation}
$$

假设配对 $V_{k,k'} $ 可写成

$$
\begin{equation}
\begin{aligned}
V_{k,k'}
=-\frac{g }{V } \gamma_k \gamma_{k'},\quad
\gamma_k\in \mathbb{R}
\end{aligned}
\end{equation}
$$

则哈密顿量化为

$$
\begin{equation}
\begin{aligned}
H
=\sum_{k,\sigma} \left(\varepsilon_k - \mu \right) c^\dag_{k\sigma} c_{k\sigma} - \frac{g }{V } \left(\sum_{k} \gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} \right) \left(\sum_{k'} \gamma_{k'} c_{-k'\downarrow} c_{k'\uparrow} \right)
\end{aligned}
\end{equation}
$$

考虑平均场近似：

$$
\begin{equation}
\begin{aligned}
&\left(\sum_{k} \gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} \right) \left(\sum_{k'} \gamma_{k'} c_{-k'\downarrow} c_{k'\uparrow} \right) \\
\approx&\left(\sum_{k} \gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} \right) \left(\sum_{k'} \gamma_{k'} \Braket{c_{-k'\downarrow} c_{k'\uparrow} } \right) + \left(\sum_{k} \gamma_k \Braket{c^\dag_{k\uparrow} c^\dag_{-k\downarrow} } \right) \left(\sum_{k'} \gamma_{k'} c_{-k'\downarrow} c_{k'\uparrow} \right) - \left(\sum_{k} \Braket{\gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} } \right) \left(\sum_{k'} \gamma_{k'} \Braket{c_{-k'\downarrow} c_{k'\uparrow} } \right) \\
\end{aligned}
\end{equation}
$$

定义超导能隙

$$
\begin{equation}
\begin{aligned}
\Delta
\equiv -\frac{g }{V } \sum_{k'} \gamma_{k'} \Braket{c_{-k'\downarrow} c_{k'\uparrow} },\quad
\Delta^*
=-\frac{g }{V } \sum_{k} \gamma_k \Braket{c^\dag_{k\uparrow} c^\dag_{-k\downarrow} }
\end{aligned}
\end{equation}
$$

则

$$
\begin{equation}
\begin{aligned}
&\left(\sum_{k} \gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} \right) \left(\sum_{k'} \gamma_{k'} c_{-k'\downarrow} c_{k'\uparrow} \right) \\
\approx&\left(\sum_{k} \gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} \right) \left(\sum_{k'} \gamma_{k'} \Braket{c_{-k'\downarrow} c_{k'\uparrow} } \right) + \left(\sum_{k} \gamma_k \Braket{c^\dag_{k\uparrow} c^\dag_{-k\downarrow} } \right) \left(\sum_{k'} \gamma_{k'} c_{-k'\downarrow} c_{k'\uparrow} \right) - \left(\sum_{k} \Braket{\gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} } \right) \left(\sum_{k'} \gamma_{k'} \Braket{c_{-k'\downarrow} c_{k'\uparrow} } \right) \\
=&\frac{-V\Delta }{g }  \left(\sum_{k} \gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} \right) + \frac{-V \Delta^* }{g }  \left(\sum_{k'} \gamma_{k'} c_{-k'\downarrow} c_{k'\uparrow} \right) - \frac{V^2 \left|\Delta \right|^2 }{g^2 }  \\
\end{aligned}
\end{equation}
$$

于是平均场哈密顿量为：

$$
\begin{equation}
\begin{aligned}
H
&=\sum_{k,\sigma} \left(\varepsilon_k - \mu \right) c^\dag_{k\sigma} c_{k\sigma} - \frac{g }{V } \left(\sum_{k} \gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} \right) \left(\sum_{k'} \gamma_{k'} c_{-k'\downarrow} c_{k'\uparrow} \right) \\
&\approx \sum_{k,\sigma} \left(\varepsilon_k - \mu \right) c^\dag_{k\sigma} c_{k\sigma} - \frac{g }{V } \left[\left(\sum_{k} \gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} \right) \left(\sum_{k'} \gamma_{k'} \Braket{c_{-k'\downarrow} c_{k'\uparrow} } \right) + \left(\sum_{k} \gamma_k \Braket{c^\dag_{k\uparrow} c^\dag_{-k\downarrow} } \right) \left(\sum_{k'} \gamma_{k'} c_{-k'\downarrow} c_{k'\uparrow} \right) - \left(\sum_{k} \Braket{\gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} } \right) \left(\sum_{k'} \gamma_{k'} \Braket{c_{-k'\downarrow} c_{k'\uparrow} } \right) \right] \\
&=\sum_{k,\sigma} \left(\varepsilon_k - \mu \right) c^\dag_{k\sigma} c_{k\sigma} - \frac{g }{V } \left[\frac{-V\Delta }{g }  \left(\sum_{k} \gamma_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} \right) + \frac{-V \Delta^* }{g }  \left(\sum_{k'} \gamma_{k'} c_{-k'\downarrow} c_{k'\uparrow} \right) - \frac{V^2 \left|\Delta \right|^2 }{g^2 } \right] \\
&=\sum_{k,\sigma} \left(\varepsilon_k - \mu \right) c^\dag_{k\sigma} c_{k\sigma} + \sum_{k} \left(\Delta \gamma_k \right) c^\dag_{k\uparrow} c^\dag_{-k\downarrow} + \sum_{k'} \left(\Delta^* \gamma_{k'} \right) c_{-k'\downarrow} c_{k'\uparrow} + \frac{V }{g } \left|\Delta \right|^2  \\
\end{aligned}
\end{equation}
$$

定义超导配对能隙函数

$$
\begin{equation}
\begin{aligned}
\Delta_k
\equiv \Delta \gamma_k,\quad
\Delta_{k'}^*
=\Delta^* \gamma_{k'}
\end{aligned}
\end{equation}
$$

定义相对于化学势的单电子能量

$$
\begin{equation}
\begin{aligned}
\xi_k
\equiv \varepsilon_k -\mu
\end{aligned}
\end{equation}
$$

最终得到BCS平均场哈密顿量：

$$
\begin{equation}
\begin{aligned}
H
=\sum_{k,\sigma} \xi_k c^\dag_{k\sigma} c_{k\sigma} + \sum_{k} \Delta_k c^\dag_{k\uparrow} c^\dag_{-k\downarrow} + \sum_{k'} \Delta_k^* c_{-k'\downarrow} c_{k'\uparrow} + \frac{V }{g } \left|\Delta \right|^2
\end{aligned}
\end{equation}
$$
