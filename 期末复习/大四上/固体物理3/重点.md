#### 无相互作用自由电子气基态能量

考虑 $N $ 个电子，则基态是费米球内全填满的态。

$$
\begin{equation}
E_{\mathrm{GS}}
=\sum_{\left|\bm{k} \right|\leqslant k_F,\sigma} \frac{k^2 }{2m } 
=\sum_\sigma \int\limits_{\left|\bm{k} \right|\leqslant k_F} \frac{\mathrm{d}^dk }{\left(2\pi \right)^d/V } \frac{k^2 }{2m } 
=2\frac{V }{\left(2\pi \right)^d 2m } \int\limits_{\left|\bm{k} \right|\leqslant k_F} \mathrm{d}^dk k^2
\end{equation}
$$

三维情况下 $d=3 $，粒子数可由下式确定：

$$
\begin{equation}
N
=\sum_\sigma \int\limits_{\left|\bm{k}\leqslant k_F \right|} \frac{\mathrm{d}^3k }{\left(2\pi \right)^3/V } 
=\frac{V }{3\pi^2 } k_F^3
\end{equation}
$$

基态能量：

$$
\begin{equation}
\begin{aligned}
E_{\mathrm{GS}}
&=2\frac{V }{\left(2\pi \right)^3 2m } \int\limits_{\left|\bm{k} \right|\leqslant k_F} \mathrm{d}^3k k^2
=2\frac{V }{\left(2\pi \right)^3 2m } \int_{k=0}^{k=k_F} 4\pi k^2 \cdot k^2 \mathrm{d}k \\
&=2\frac{V }{\left(2\pi \right)^3 2m } \cdot 4\pi \frac{k_F^5 }{5 } 
=\frac{V }{10\pi^2 m } k_F^5
=\frac{3 }{5 } N \varepsilon_F
\end{aligned}
\end{equation}
$$

#### HF近似算 $H_{\mathrm{int}} $

Hartree-Fock近似（平均场近似）

$$
\begin{equation}
A B 
\approx \Braket{A } B + A \Braket{B } - \Braket{A }\Braket{B }
\end{equation}
$$

这里 $\Braket{ } $ 代表系综平均或基态期望。

对于Coulomb电子气，对相互作用项中的四费米子项用H-F近似：

$$
\begin{equation}
H_{\mathrm{int}}
=\frac{1 }{2 V} \sum_{k,k',q\ne 0,\sigma,\sigma'} \frac{4\pi e^2 }{q^2 }  c_{k+q,\sigma}^\dag c_{k'-q,\sigma'}^\dag c_{k',\sigma'} c_{k,\sigma}
\end{equation}
$$

$$
\begin{equation}
\begin{aligned}
&c_{k+q,\sigma}^\dag c_{k'-q,\sigma'}^\dag c_{k',\sigma'} c_{k,\sigma} \\
=&-c_{k+q,\sigma}^\dag c_{k',\sigma'} c_{k'-q,\sigma'}^\dag c_{k,\sigma} \\
\approx& -\Braket{c_{k+q,\sigma}^\dag c_{k',\sigma'} } c_{k'-q,\sigma'}^\dag c_{k,\sigma} - c_{k+q,\sigma}^\dag c_{k',\sigma'} \Braket{c_{k'-q,\sigma'}^\dag c_{k,\sigma} } + \Braket{c_{k+q,\sigma}^\dag c_{k',\sigma'} } \Braket{c_{k'-q,\sigma'}^\dag c_{k,\sigma} } \\
=&-\delta_{k',k+q} \delta_{\sigma,\sigma'} \Braket{c_{k',\sigma'}^\dag c_{k',\sigma'} } c_{k'-q,\sigma'}^\dag c_{k,\sigma} - \delta_{k,k'-q} \delta_{\sigma,\sigma'} \Braket{c_{k,\sigma}^\dag c_{k,\sigma} } c_{k+q,\sigma}^\dag c_{k',\sigma'} + \delta_{k',k+q} \delta_{\sigma,\sigma'} \delta_{k,k'-q} \Braket{c_{k',\sigma'}^\dag c_{k',\sigma'} } \Braket{c_{k,\sigma}^\dag c_{k,\sigma} } \\
=&-\delta_{k',k+q} \delta_{\sigma,\sigma'} \Braket{c_{k',\sigma'}^\dag c_{k',\sigma'} } c_{k,\sigma}^\dag c_{k,\sigma} - \delta_{k,k'-q} \delta_{\sigma,\sigma'} \Braket{c_{k,\sigma}^\dag c_{k,\sigma} } c_{k',\sigma'}^\dag c_{k',\sigma'} + \delta_{k',k+q} \delta_{\sigma,\sigma'} \delta_{k,k'-q} \Braket{c_{k',\sigma'}^\dag c_{k',\sigma'} } \Braket{c_{k,\sigma}^\dag c_{k,\sigma} } \\
\end{aligned}
\end{equation}
$$

忽略最后的常数项就有

$$
\begin{equation}
\begin{aligned}
c_{k+q,\sigma}^\dag c_{k'-q,\sigma'}^\dag c_{k',\sigma'} c_{k,\sigma}
&\approx -\delta_{k',k+q} \delta_{\sigma,\sigma'} \Braket{c_{k',\sigma'}^\dag c_{k',\sigma'} } c_{k,\sigma}^\dag c_{k,\sigma} - \delta_{k,k'-q} \delta_{\sigma,\sigma'} \Braket{c_{k,\sigma}^\dag c_{k,\sigma} } c_{k',\sigma'}^\dag c_{k',\sigma'} \\
&=-\delta_{k',k+q} \delta_{\sigma,\sigma'} n_{k',\sigma'} c_{k,\sigma}^\dag c_{k,\sigma} - \delta_{k,k'-q} \delta_{\sigma,\sigma'} n_{k,\sigma} c_{k',\sigma'}^\dag c_{k',\sigma'} \\
&=-\delta_{k',k+q} \delta_{\sigma,\sigma'} \theta(k_F-k') c_{k,\sigma}^\dag c_{k,\sigma} - \delta_{k,k'-q} \delta_{\sigma,\sigma'} \theta(k_F-k) c_{k',\sigma'}^\dag c_{k',\sigma'} \\
\end{aligned}
\end{equation}
$$

则

$$
\begin{equation}
\begin{aligned}
H_{\mathrm{int}}
&=\frac{1 }{2 V} \sum_{k,k',q\ne 0,\sigma,\sigma'} \frac{4\pi e^2 }{q^2 }  c_{k+q,\sigma}^\dag c_{k'-q,\sigma'}^\dag c_{k',\sigma'} c_{k,\sigma} \\
&\approx \frac{1 }{2 V} \sum_{k,k',q\ne 0,\sigma,\sigma'} \frac{4\pi e^2 }{q^2 } \left[-\delta_{k',k+q} \delta_{\sigma,\sigma'} \theta(k_F-k') c_{k,\sigma}^\dag c_{k,\sigma} - \delta_{k,k'-q} \delta_{\sigma,\sigma'} \theta(k_F-k) c_{k',\sigma'}^\dag c_{k',\sigma'} \right] \\
&=-\frac{1 }{2V } \sum_{k,k',\sigma} \frac{4\pi e^2 }{\left|k'-k \right|^2 } \left[\theta(k_F-k') c_{k,\sigma}^\dag c_{k,\sigma} + \theta(k_F-k) c_{k',\sigma}^\dag c_{k',\sigma} \right] \\
&=-\frac{1 }{2V } \sum_{k,k',\sigma} \frac{4\pi e^2 }{\left|k'-k \right|^2 } \left[\theta(k_F-k') c_{k,\sigma}^\dag c_{k,\sigma} + \theta(k_F-k') c_{k,\sigma}^\dag c_{k,\sigma} \right] \\
&=-\frac{1 }{V } \sum_{k,k',\sigma} \frac{4\pi e^2 }{\left|k'-k \right|^2 } \theta(k_F-k') c_{k,\sigma}^\dag c_{k,\sigma} \\
\end{aligned}
\end{equation}
$$

#### 两自旋模型本征解

两自旋-1/2Heisenberg耦合：

$$
\begin{equation}
H
=J\vec{S}_1\cdot\vec{S}_2
\end{equation}
$$

定义总自旋和总自旋 $z $ 分量：

$$
\begin{equation}
\vec{S}\equiv \vec{S}_1 + \vec{S}_2,\quad
S^z\equiv S_1^z + S_2^z
\end{equation}
$$

利用

$$
\begin{equation}
\vec{S}^2=\vec{S}_1^2 + \vec{S}_2^2 + 2\vec{S}_1\cdot\vec{S}_2,\quad
\vec{S}_1\cdot\vec{S}_2=\frac{1 }{2 } \left(\vec{S}^2 - \vec{S}_1^2-\vec{S}_2^2 \right)
=\frac{1 }{2 } \left(\vec{S}^2 - \frac{3 }{2 }  \right)
\end{equation}
$$

其中 $\vec{S}_1^2,\vec{S}_2^2 $ 是常数，可以证明

$$
\begin{equation}
\left[H , \vec{S}^2 \right] = 0,\quad
\left[H , S^z \right] = 0,\quad
\left[\vec{S}^2 , S^z \right] = 0
\end{equation}
$$

因此 $\vec{S}^2,S^z $ 的共同本征态（自旋单态、自旋三重态）也是 $H $ 本征态。

$$
\begin{equation}
\begin{aligned}
H \left[\frac{1 }{\sqrt{2} } \left(\Ket{\uparrow \downarrow } - \Ket{\downarrow \uparrow} \right) \right]
&=J \frac{1 }{2 } \left(\vec{S}^2 - \frac{3 }{2 }  \right) \frac{1 }{\sqrt{2} } \left(\Ket{\uparrow \downarrow } - \Ket{\downarrow \uparrow} \right)
=J \frac{1 }{2 } \left(0(0+1) - \frac{3 }{2 }  \right) \frac{1 }{\sqrt{2} } \left(\Ket{\uparrow \downarrow } - \Ket{\downarrow \uparrow} \right) \\
&=-\frac{3 }{4 } J \frac{1 }{\sqrt{2} } \left(\Ket{\uparrow \downarrow } - \Ket{\downarrow \uparrow} \right)
\end{aligned}
\end{equation}
$$

$$
\begin{equation}
\begin{aligned}
H \Ket{\mathrm{triplet} }
&=J \frac{1 }{2 } \left(\vec{S}^2 - \frac{3 }{2 }  \right) \Ket{\mathrm{triplet} }
=J \frac{1 }{2 } \left(1(1+1) - \frac{3 }{2 }  \right) \Ket{\mathrm{triplet} }
=\frac{J }{4 } \Ket{\mathrm{triplet} }
\end{aligned}
\end{equation}
$$
