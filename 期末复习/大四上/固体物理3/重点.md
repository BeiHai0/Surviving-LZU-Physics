#### 无相互作用自由电子气基态能量

考虑 $N $ 个电子，则基态是费米球内全填满的态。

$$
\begin{equation}
E_{\mathrm{GS}}
=\sum_{\left|\bm{k} \right|\leqslant k_F,\sigma} \frac{k^2 }{2m } 
=\sum_\sigma \int\limits_{\left|\bm{k} \right|\leqslant k_F} \frac{\mathrm{d}^dk }{\left(2\pi \right)^d/V } \frac{k^2 }{2m } 
=2\frac{V }{\left(2\pi \right)^d 2m } \int\limits_{\left|\bm{k} \right|\leqslant k_F} \mathrm{d}^dk k^2
\end{equation}
$$

三维情况下 $d=3 $，粒子数可由下式确定：

$$
\begin{equation}
N
=\sum_\sigma \int\limits_{\left|\bm{k}\leqslant k_F \right|} \frac{\mathrm{d}^3k }{\left(2\pi \right)^3/V } 
=\frac{V }{3\pi^2 } k_F^3
\end{equation}
$$

基态能量：

$$
\begin{equation}
\begin{aligned}
E_{\mathrm{GS}}
&=2\frac{V }{\left(2\pi \right)^3 2m } \int\limits_{\left|\bm{k} \right|\leqslant k_F} \mathrm{d}^3k k^2
=2\frac{V }{\left(2\pi \right)^3 2m } \int_{k=0}^{k=k_F} 4\pi k^2 \cdot k^2 \mathrm{d}k \\
&=2\frac{V }{\left(2\pi \right)^3 2m } \cdot 4\pi \frac{k_F^5 }{5 } 
=\frac{V }{10\pi^2 m } k_F^5
=\frac{3 }{5 } N \varepsilon_F
\end{aligned}
\end{equation}
$$

#### 两自旋模型本征解

两自旋-1/2Heisenberg耦合：

$$
\begin{equation}
H
=J\vec{S}_1\cdot\vec{S}_2
\end{equation}
$$

定义总自旋和总自旋 $z $ 分量：

$$
\begin{equation}
\vec{S}\equiv \vec{S}_1 + \vec{S}_2,\quad
S^z\equiv S_1^z + S_2^z
\end{equation}
$$

利用

$$
\begin{equation}
\vec{S}^2=\vec{S}_1^2 + \vec{S}_2^2 + 2\vec{S}_1\cdot\vec{S}_2,\quad
\vec{S}_1\cdot\vec{S}_2=\frac{1 }{2 } \left(\vec{S}^2 - \vec{S}_1^2-\vec{S}_2^2 \right)
=\frac{1 }{2 } \left(\vec{S}^2 - \frac{3 }{2 }  \right)
\end{equation}
$$

其中 $\vec{S}_1^2,\vec{S}_2^2 $ 是常数，可以证明

$$
\begin{equation}
\left[H , \vec{S}^2 \right] = 0,\quad
\left[H , S^z \right] = 0,\quad
\left[\vec{S}^2 , S^z \right] = 0
\end{equation}
$$

因此 $\vec{S}^2,S^z $ 的共同本征态（自旋单态、自旋三重态）也是 $H $ 本征态。

$$
\begin{equation}
\begin{aligned}
H \left[\frac{1 }{\sqrt{2} } \left(\Ket{\uparrow \downarrow } - \Ket{\downarrow \uparrow} \right) \right]
&=J \frac{1 }{2 } \left(\vec{S}^2 - \frac{3 }{2 }  \right) \frac{1 }{\sqrt{2} } \left(\Ket{\uparrow \downarrow } - \Ket{\downarrow \uparrow} \right)
=J \frac{1 }{2 } \left(0(0+1) - \frac{3 }{2 }  \right) \frac{1 }{\sqrt{2} } \left(\Ket{\uparrow \downarrow } - \Ket{\downarrow \uparrow} \right) \\
&=-\frac{3 }{4 } J \frac{1 }{\sqrt{2} } \left(\Ket{\uparrow \downarrow } - \Ket{\downarrow \uparrow} \right)
\end{aligned}
\end{equation}
$$

$$
\begin{equation}
\begin{aligned}
H \Ket{\mathrm{triplet} }
&=J \frac{1 }{2 } \left(\vec{S}^2 - \frac{3 }{2 }  \right) \Ket{\mathrm{triplet} }
=J \frac{1 }{2 } \left(1(1+1) - \frac{3 }{2 }  \right) \Ket{\mathrm{triplet} }
=\frac{J }{4 } \Ket{\mathrm{triplet} }
\end{aligned}
\end{equation}
$$

#### 自旋波的来历（HP bosons）

一维最近邻耦合铁磁量子Heisenberg模型模型：

$$
\begin{equation}
H
=-J \sum_j \vec{S}_j\cdot\vec{S}_{j+1}
=-J\sum_j \left(S_{j}^x S_{j+1}^x + S_{j}^y S_{j+1}^y + S_{j}^z S_{j+1}^z \right)
\end{equation}
$$

引入升降算符

$$
\begin{equation}
S_j^\pm 
\equiv S_j^x \pm \mathrm{i} S_j^y
\end{equation}
$$

改写哈密顿量

$$
\begin{equation}
H_{FM}
=-J\sum_j \left[\frac{1 }{2 } \left(S_j^+ S_{j+1}^- + S_j^- S_{j+1}^+ \right) + S_j^z S_{j+1}^z \right]
\end{equation}
$$

考虑周期边界条件，可以发现铁磁态（所有自旋都朝一个方向）是体系的本征态，并且是基态。

$$
\begin{equation}
\Ket{\psi_{FM} }
\equiv \Ket{\uparrow }_1\cdots \Ket{\uparrow }_{N_s}
\end{equation}
$$

$$
\begin{equation}
\begin{aligned}
H_{FM} \Ket{\psi_{FM} }
=-\frac{J }{4 } N_s \Ket{\psi_{FM} }
\end{aligned}
\end{equation}
$$

考虑完全极化铁磁态基础上，某个自旋翻转得到的态：

$$
\begin{equation}
\Ket{\psi_i }
\equiv \Ket{\downarrow }_i \prod_{j\ne i} \Ket{\uparrow }_j
\end{equation}
$$

这个态不是哈密顿量本征态。可以发现

$$
\begin{equation}
H_{FM} \Ket{\psi_i }
=-\frac{J }{2 } \left(\Ket{\psi_{i-1} } + \Ket{\psi_{i+1} } \right) + \left(-\frac{J }{4 } \left(N_s-2 \right) + \frac{J }{4 } \cdot 2 \right) \Ket{\psi_i }
\end{equation}
$$

因此构造如下的态是体系哈密顿量本征态：

$$
\begin{equation}
\Ket{\psi }
\equiv \sum_i c_i \Ket{\psi_i }
\end{equation}
$$

若所有的系数 $c_i $ 都不为零，则这个态描述的是在晶格自旋体系中传播的自旋翻转。

利用本征方程

$$
\begin{equation}
H_{FM} \Ket{\psi } = E \Ket{\psi }
\end{equation}
$$

可以求得：

$$
\begin{equation}
\Ket{\psi }
=\sum_i \mathrm{e}^{\mathrm{i} k R_i} \Ket{\psi_i }
\end{equation}
$$

$$
\begin{equation}
E
=E_{FM} + J(1-\cos k)
\end{equation}
$$

其中 $R_i $ 是格点 $i $ 的位置，这就是铁磁自旋波与其激发谱。

=\frac{1 }{\sqrt{N_s/2} } \sum_k \mathrm{e}^{\mathrm{i}kR_i} a_k,\quad

\begin{equation}

\end{equation}
$$

则

$$
\begin{equation}
\begin{aligned}
H_{AF}
=JS\sum_k \left[Z a_k^\dag a_k  + Z b_k^\dag b_k + \left(\sum_\delta \mathrm{e}^{\mathrm{i}k\delta} \right) a_k^\dag b_k^\dag + \left(\sum_\delta \mathrm{e}^{-\mathrm{i}k\delta} \right) a_k b_k \right] - J N_s \frac{Z }{2 } S^2
\end{aligned}
\end{equation}

$$
\begin{equation}
\begin{aligned}
H_{AF}

\end{aligned}
\end{equation}
$$

$$
\begin{equation}
\begin{pmatrix}
a_k \\
b_k^\dag 
\end{pmatrix}
=\begin{pmatrix}
\cosh \theta_k &-\sinh\theta_k \\
-\sinh\theta_k &\cosh\theta_k
\end{pmatrix}
\begin{pmatrix}
\alpha_k \\
\beta_k^\dag
\end{pmatrix}
\end{equation}
$$

为了消去非对角项，取

$$
\begin{equation}
\gamma_k = \tanh 2\theta_k
\end{equation}
$$

则哈密顿量化为

$$
\begin{equation}
H_{AF}
=JSZ \sum_k \left(\cosh 2\theta_k - \gamma_k \sinh 2\theta_k \right)\left(\alpha_k^\dag \alpha_k + \beta^\dag \beta_k \right) + JSZ\sum_k \left(\cosh 2\theta_k - \gamma_k \sinh 2\theta_k \right)
\end{equation}
$$

$$
\begin{equation}

=\sum_k \omega_k\left(\alpha_k^\dag \alpha_k + \beta^\dag \beta_k \right) + \sum_k \omega_k - J N_s \frac{Z }{2 } S(S+1)
\end{equation}
$$

$$
\begin{equation}
\omega_k = JSZ \sqrt{1-\gamma_k^2}
\end{equation}
$$

### ch4

#### Landau-Fermi液体是什么

Landau费米液体的基本结果：电子-电子相互作用的唯一效果是改变电子的质量，对物理可观测量给出小的修正并导致无碰撞的零声波。

费米液体的两个基本假设：

绝热性：无相互作用费米气体在缓慢加入相互作用后变成费米液体；

一对一：费米子液体的准粒子/空穴激发与费米气体的粒子/空穴一一对应。

#### 零声是什么

对于波动解 $n_{k\sigma}(x,t)=f_F\left(\varepsilon^0_{k\sigma} \right) - \mathrm{e}^{\mathrm{i}(qx-\omega t)} \eta\left(\hat{k} \right)\delta\left(\varepsilon^0_{k\sigma} \right) $，在 $q\to 0 $ 的长波极限下，粒子数分布的变化 $\delta n_{k\sigma}(t)=-\mathrm{e}^{-\mathrm{i}\omega t} \eta\left(\hat{k} \right) \delta(\varepsilon) $，在物理上可以理解为，若以平衡时的费米球为参考，$\delta n_{k\sigma}(t) $ 描述在费米子面 $\delta(\varepsilon) $ 附近，体系以频率 $\omega $ 放大或缩小费米球，这种振动就是零声。零声模式来自分布函数在时空上的振荡行为，可以看作是费米液体中的费米球的集体运动。零声模式是无能隙的低能激发。

#### Landau相变理论是什么

chat：用一个序参量来区分不同相，并在满足体系对称性的前提下，将自由能按序参量作低阶展开。随温度变化，自由能最低点从零序参量转移到非零值，从而描述自发对称性破缺和连续（二级）相变。该理论忽略涨落效应，在临界点附近或低维体系中需要修正。

#### 序参量是什么

chat：序参量（order parameter）是一个在高对称相中为零、在低对称相中非零的物理量，用来区分不同相并刻画自发对称性破缺。在 Landau 相变理论中，自由能被写成序参量的函数，通过其最小值随温度变化来描述相变。

#### Anderson-Higgs机制是什么

chat：当一个体系中连续的全局对称性发生自发破缺时，本来应该出现的 Goldstone 无质量模，如果该对称性同时被局域化并与 gauge 场耦合，那么这个 Goldstone 模式就不再作为独立的低能激发存在，而是被 gauge 场吸收，使得原本无质量的 gauge 玻色子获得一个质量。

#### Goldstone是什么

chat：当体系的连续全局对称性在基态中被自发破缺时，序参量在破缺方向上的相位涨落不会增加能量，从而产生能隙为零（或低能）的集体激发，这种激发称为 Goldstone 模式。

#### 破缺了什么对称性？

光子不破缺U(1)对称性，声子不破缺U(1)，声子破缺空间（连续）平移对称性。

| 体系          | 破缺的连续对称性                   | Goldstone 模式     |
| ----------- | -------------------------- | ---------------- |
| 晶体          | 空间**连续平移**                 | 声子（phonon）       |
| 各向同性铁磁体     | **自旋旋转** $SO(3)\to SO(2)$  | 自旋波（magnon）      |
| 反铁磁体        | 自旋旋转 $SO(3)\to SO(2)$      | 自旋波（两支）          |
| 中性超流（He-4）  | 全局 $U(1)$                  | 相位声子             |
| BEC         | 全局 $U(1)$                  | 相位模              |
| 液晶（向列相）     | 空间旋转                       | director 波       |
| 手征对称破缺（QCD） | 手征 $SU(N)_L\times SU(N)_R$ | 赝 Goldstone（π介子） |

$$

$$

### ch15

#### Z2规范理论是什么

规范自由度只取两个值（±1），规范群是 $Z_2 $ 的晶格规范理论。

#### 规范会变的物理量的期望值

局域规范对称性不会自发破缺。非规范不变量平均值为零。只有规范不变量才会有非零的期望值。

#### Wilson loop

Wilson loop是以自旋构成的非局域序参量：

$$
\begin{equation}
W(C)
=\prod_{i,j\in C} \sigma_{i,j}
\end{equation}
$$

#### 禁闭、退禁闭

高温下满足面积律的相叫做规范场的禁闭态，低温下满足周长律的相叫做退禁闭态。

当规范场处于禁闭态，往体系中放两个例子，则粒子与规范场耦合形成束缚态，不能单独运动。

而周长 $E^{-P_c}\sim\mathrm{e}^{-T}\sim \mathrm{e}^{-V(R) T} $，势能与 $R $ 无关，电荷可以自由运动不受限制。

#### 规范变换的面积律、周长律

面积律：在高温极限下，Wilson loop的平均值按具有指数型的按面积衰减形式

$$
\begin{equation}
\Braket{W(C) } \sim \mathrm{e}^{-A_c}
\end{equation}
$$

周长律：低温时， $Z_2 $ 规范理论的Wilson loop具有指数型的按回路周长衰减形式

$$
\begin{equation}
\Braket{W(C) } \sim \mathrm{e}^{-P_c}
\end{equation}
$$

#### toric-code模型

$$
\begin{equation}
\begin{aligned}
H
&=-J \sum_i \sigma_{i,i+\hat{x}}^z \sigma_{i+\hat{x},i+\hat{x}+\hat{y}}^z \sigma_{i+\hat{x}+\hat{y},i+\hat{y}}^z \sigma_{i+\hat{y},i}^z - w\sum_i G(i) \\
&=-J_e \sum_i A_s(i) - J_m \sum_i B_p(i)
\end{aligned}
\end{equation}
$$

$B_p(i) $ 代表格点 $i $ 格点对应小正方形上 $\sigma^z $ 的乘积，$A_s(i) $ 代表以 $i $ 格点为中心所有最近邻 $\sigma^x $ 的乘积（形成一个star）

#### 如何在六角晶格上构造model

#### 基态波函数

对于电荷激发可以定义

$$
\begin{equation}
W^{(e)}_l
\equiv \prod_{(i,j)\in l} \sigma_{i,j}^z
\end{equation}
$$

其物理意义是在几天上产生两个电荷激发。

也可定义产生涡旋的String算符

$$
\begin{equation}
W^{(m)}_l
\equiv \prod_{(i,j)\in l}\sigma_{i,j}^x
\end{equation}
$$

对于定义在轮胎面上的正方晶格，利用闭合的String算符可以构造四个拓扑不等价的基态

$$
\begin{equation}
\Ket{\Psi_0 },W_{C_x}^{(m)}\Ket{\Psi_0 },W^{(e)}_{C_y}\Ket{\Psi_0 },W_{C_x}^{(m)}W^{(e)}_{C_y}\Ket{\Psi_0 }
\end{equation}
$$

对于开边界情况，基态可以由投影算符得到

$$
\begin{equation}
\Ket{\Psi_0 }
=\prod_p \frac{1+B_p }{2 } \Ket{0 }
\end{equation}
$$

其中 $\Ket{0 }\equiv \Ket{\rightarrow\rightarrow\cdots } $ 是所有自旋 $\sigma^x $ 本征态为 $1 $ 的乘积态。
