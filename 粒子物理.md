## 粒子的质量

速度为 $\vec{v} $ 运动的粒子，其能量 $E $，动量 $\vec{p} $ 和静止质量 $m $ 满足：

$$
E=\frac{m }{\sqrt{1-\vec{v}^2} } ,\quad
\vec{p}=\frac{m\vec{v} }{\sqrt{1-\vec{v}^2} } ,\quad
E^2 - \vec{p}^2 = m^2,(质壳条件)
$$

实验上不稳定粒子的质量是一个几率分布（此处有图）：

$$
\rho(M)
=\frac{\Gamma }{2\pi\left[\left(M-m \right)^2+\Gamma^2/4 \right] } 
$$

$m $ 称为粒子的质量，$\Gamma $ 称为粒子的宽度。

质量 $m $ 的物理意义：最可几取值；宽度 $\Gamma $ 的物理意义：质量分布半高宽。

对于一个质量为 $m $，宽度为 $\Gamma $ 的粒子，该粒子质量测量值在 $\left[m-\Gamma/2,m+\Gamma/2 \right] $ 区间的概率为 $0.5 $，质量测量值小于 $m-\Gamma/2 $ 或大于 $m+\Gamma/2 $ 的概率各为 $0.25 $。

### 一些粒子的质量

光子质量：接近零；电子质量：$0.51~\mathrm{MeV} $；质子质量：$938~\mathrm{MeV} $

## 粒子的寿命

粒子的寿命是粒子静止时的平均寿命。粒子产生后到衰变前存在的时间就是该粒子的寿命。粒子的寿命是大量粒子寿命的平均值，都是值粒子静止时的寿命。

粒子寿命 $\tau $ 和宽度 $\Gamma $ 满足：$\Gamma \tau = 1.$

分支比：不稳定粒子衰变可以有不同的衰变方式。每种衰变方式所占比例称为该衰变方式的分支比 $R_i .$ 所有衰变方式分支比之和为 $1 .$

寿命测量方式：若粒子寿命长，可通过粒子衰变前径迹长度确定寿命；若粒子寿命短，可通过测量粒子质量分布，量出宽度 $\Gamma $，再根据 $\Gamma \tau=1 $ 计算寿命。

粒子的宽度 $\Gamma $ 是粒子寿命的倒数，它的概率含义是单位时间内粒子衰变掉的概率。单位时间内粒子衰变到第 $i $ 衰变道的概率称为该衰变道的部分宽度 $\Gamma_i .$

近域共振态：粒子的某一衰变道的末态各粒子的质量总和大于该不稳定粒子的质量，这样的粒子称为近域共振态。$f_0(980) $ 粒子是典型的近域共振态。$\displaystyle{\frac{\Gamma\left(f_0(980) \to K^+ K^- \right) }{\Gamma\left(f_0\left(980 \right)\to \pi^+ \pi^- \right) } = 0.69\pm 0.32  }$，$f_0(980) $ 粒子质量中心值为 $980~\mathrm{MeV} $，大于末态 $K \bar{K} $ 的质量和。这是因为 $f_0(980) $ 的质量是一个几率分布，质量分布的“尾巴”部分可以衰变到 $K \bar{K} $ 末态。

## 三、粒子的运动学描述

### 粒子的能量和动量

$E^2-\vec{p}^2 = m^2,\vec{v} = \frac{\vec{p} }{E } = \frac{\vec{p} }{\sqrt{m^2+\vec{p}^2} } $；若粒子质量为零，则 $v = 1$，即光速。

粒子动量可按纵动量（一维）和横动量（二维）进行分解。

把空间某一方向作为标准方向，可把粒子动量分解为纵动量 $p_L $ 和横动量 $\vec{p}_T $ 之和。若沿这个标准方向作 Lorentz 变化，则 $p_L'= \gamma(p_L-\beta E),E'=\gamma(E-\beta p_L),\gamma=1/\sqrt{1-\beta^2} $，$\beta $ 是两个参考系相对速度。横动量不变 $\vec{p}_T'=\vec{p}_T$

横能量：$E_T = E\sin\theta$，$\theta $ 是粒子运动方向与标准方向的夹角。$E_T=\sqrt{\left(\vec{p}^2 + m^2 \right)\sin^2\theta}=\sqrt{\left(\vec{p}\sin\theta \right)\cdot\left(\vec{p}\sin\theta \right) +m^2\sin^2\theta}=\sqrt{\vec{p}_T^2 + m^2\sin^2\theta} .$ 由于横动量 $\vec{p}_T $ 在 Lorentz 变换下不变，因此不同参考系中的横能量与 $\theta $ 有关。

$$
E
=\sqrt{m^2 + \frac{\vec{p}_T^2 }{\sin^2\theta } }
$$

### 粒子的快度和赝快度

双曲正弦：$\mathrm{sh}(x)\equiv \left[\exp(x) - \exp(-x) \right]/2 $

双曲余弦：$\mathrm{ch}(x)\equiv \left[\exp(x)+ \exp(-x) \right]/2 $

双曲正切：$\mathrm{th}(x)\equiv \mathrm{sh}(x)/\mathrm{ch}(x) = \left[\exp(x) - \exp(-x) \right]/\left[\exp(x)+ \exp(-x) \right] $

$v_L $：速度在标准方向的投影。

快度 $y $ 满足：$\th y = v_L,\quad \sh y = v_L/\sqrt{1-v_L^2} ,\quad\ch y = 1/\sqrt{1-v_L^2} $

粒子的快度 $y $ 是速度投影 $v_L $ 的单调递增函数。当 $v_L $ 从 $-1 $ 增加到 $0 $ 再增加到 $1 $ 时，快度 $y $ 相应从 $-\infty $ 增加到 $0 $ 再增加到 $\infty .$

当 $v_L\to 0 $ 时，有近似公式 $y\approx v_L $

在 Lorentz 变换下，快度的变换规律：$y'=y-y_0 $，$y_0 $ 为两个参考系之间的快度。

粒子的快度用能量、动量表示：

$$
y = \frac{1 }{2 } \ln\frac{E+p_L }{E-p_L } ,\quad
\th y = \frac{p_L }{E } 
$$

纵动量 $p_L $ 与能量和快度的关系：

$$
p_L = m_T\sh y,\quad
E = m_T \mathrm{ch}~y
$$

横质量 $m_T=\sqrt{m^2+p_T^2} $，其沿纵向的 Lorentz 变换下不变。

赝快度 $\eta $：$\eta = -\ln\tan\frac{\theta }{2 } $，$\theta $ 为粒子运动方向与标准方向的夹角。

赝快度和快度的关系：

$$
y = \th^{-1}\left[\frac{1 }{\sqrt{1+m^2/p^2} } \th \eta \right]
$$

若粒子的质量远小于粒子的横动量，即 $m\ll p_T $，则有近似：

$$
y
\approx \eta - \frac{m^2 }{2 p_T^2 } \cos\theta
$$

# 第2章 对称性和守恒定律

## 对称和对称性

对称性：若某一现象或系统在某一变换下不变，则称该现象或系统具有该变换所对应的对称性。

内部对称性：物理学中独立于空间和时间的其他性质的变换所体现的对称性。



## 守恒量的一般性质

### 守恒量的分类

有经典对应的守恒量：能量、动量、角动量、电荷。

无经典对应的守恒量：同位旋、奇异数、粲数、底数、轻子数、重子数、P宇称、C宇称、G宇称。

相加性守恒量：一个复合体的总守恒量是其各个组分所贡献该守恒量的总和。能量、动量、角动量、电荷、同位旋、奇异数、粲数、底数、轻子数、重子数。

相乘性守恒量：一个复合体的总守恒量是其各个组分所贡献该守恒量的乘积。P宇称、C宇称、G宇称。

严格守恒：守恒定律对各种相互作用的成立。

近似守恒：守恒定律对某些相互作用成立，但对另一些相互作用不成立，且运动过程中后者影响是次要的。

能量、动量、角动量、电荷：有经典对应的相加性严格守恒量。

同位旋、奇异数：无经典对应的相加性近似守恒量。

P宇称、C宇称、G宇称：无经典对应的相乘性近似守恒量。

### 诺特定理

### 经典物理中的诺特定理

若运动规律在某一不明显依赖于时间的变换下具有不变性，则必定存在一个守恒定律。

变分原理：

$$
\delta S
=\int \delta L\left(q_i,\dot{q}_i \right)\mathrm{d}t
=0
$$

运动方程：

$$
\frac{\partial L }{\partial q_i } 
=\frac{\mathrm{d} }{\mathrm{d}t } \frac{\partial L }{\partial \dot{q}_i } 
$$

若运动规律在某一不明显依赖于时间的连续变换（由连续参数 $\xi $ 描写）下不变，即

$$
\delta S
=0
=\int \sum_i \left(\frac{\partial L }{\partial q_i } \frac{\partial q_i }{\partial \xi } + \frac{\partial L }{\partial \dot{q}_i } \frac{\partial \dot{q}_i }{\partial \xi }  \right) \delta \xi \mathrm{d}t
=\int \frac{\mathrm{d} }{\mathrm{d}t } \sum_i \left(\frac{\partial L }{\partial \dot{q}_i } \frac{\partial q_i }{\partial \xi } \right) \delta\xi\mathrm{d}t
$$

则有

$$
\frac{\mathrm{d} }{\mathrm{d}t } \sum_i \left(\frac{\partial L }{\partial \dot{q}_i } \frac{\partial q_i }{\partial \xi }  \right)
=0
$$

也就是说，守恒量为：

$$
F
\equiv \sum_i \left(\frac{\partial L }{\partial \dot{q}_i } \frac{\partial q_i }{\partial \xi }  \right)
$$

### 量子力学中的诺特定理

在量子力学中，任一系统的运动规律由系统的哈密顿量 $H $ 描写。若系统哈密顿量在某一连续变换下不变，则必定存在一个守恒量。

#### Ehrenfest 定理

Ehrenfest 定理给出：

$$
\frac{\mathrm{d}\braket{A} }{\mathrm{d}t } 
=\frac{1 }{\mathrm{i}\hbar } \braket{[A,H]} + \Braket{\frac{\partial A }{\partial t } }
$$

这就是说，一个不显含时间 $t $ 的物理量是守恒量的充要条件是该物理量与 $H $ 对易。

#### 连续变换

对于一个连续变换，总可引入一个连续参数 $\xi $ 来描写。$H=H(\xi) .$

运动规律在连续变换下不变意味着：

$$
H(\xi+\mathrm{d}\xi)
=H(\xi)
$$

展开至一阶小量得到：

$$
\frac{\partial H }{\partial \xi } 
=0
$$

对于任意态 $\Ket{\psi } $，有

$$
\frac{\partial }{\partial \xi } \left(H \Ket{\psi } \right)
=\frac{\partial H }{\partial \xi } \Ket{\psi } + H\frac{\partial }{\partial \xi } \Ket{\psi }
=H\frac{\partial }{\partial \xi } \Ket{\psi }
$$

也即：

$$
\left[H , \frac{\partial }{\partial \xi }  \right]
=0
$$

定义厄米算符：

$$
p
=-\mathrm{i}\frac{\partial }{\partial \xi } 
$$

则 $p $ 是守恒量。

#### 分立变换

分立变换：$H\to H'= U H U^{-1} $

若哈密顿量在分立变换下不变，即

$$
H = H' = U H U^{-1}
$$

则

$$
H U = U H
$$

即 $U $ 是守恒量。

## 五、奇异数和重子数

奇异粒子的特征：协同产生，独立衰变；快产生，慢衰变。

### 奇异数

奇异数只能取整数。

普通粒子的奇异数为 $0 $，奇异粒子的奇异数由以下反应规定：

$$
\pi^- p
\to \Lambda \pi^- K^+
$$

规定 $K^+ $ 粒子的奇异数为 $+1 $，$\Lambda $ 的奇异数为 $-1 $，然后由其他反应确定其余粒子的奇异数。

奇异数 $S=+1 $ 的奇异粒子：$K^0,K^+ $

奇异数 $S=-1 $ 的奇异粒子：$K^-,\Lambda,\Sigma^+,\Sigma^0,\Sigma^- $

奇异数 $S=-2 $ 的奇异粒子：$\Xi^0,\Xi^- $

奇异数 $S=-3 $ 的奇异粒子：$\Omega^- $

在强相互作用中，奇异粒子协同产生，奇异数 $S $ 严格守恒：

$$
\Delta S = 0
$$

弱相互作用中，奇异数 $S $ 可以不守恒：

$$
\Delta S = 0,\pm 1
$$

### 重子数

用重子数 $b $ 来区别重子中的正反粒子。

$$
b
=\frac{1 }{3 } \left(n_q-n_{\bar{q}} \right)
$$

其中，$n_q $ 表示强子内部正夸克的数目，$n_{\bar{q}} $ 表示强子内部反夸克的数目。

重子：$b=+1 $；反重子：$b=-1 $；重子以外的粒子：$b=0 $

在相互作用中重子数守恒。

重子数守恒绝对了重子的稳定性。重子数守恒决定了质子不能衰变成一个正电子和一个光子。

重子数可用于区分电中性的正反中子。中子的重子数为 $1 $，反中子的重子数为 $-1 .$

重子数是一个严格的内部相加性守恒量。

## 正反粒子变换

### C变换

$$
C\ket{A}
=C'(A) \Ket{\bar{A}}
$$

$C'(A) $ 是一个绝对值为 $1 $ 的数，称为态 $\Ket{A } $ 的C变换因子。

对于由多个粒子构成的系统，C变换可表示为：

$$
C \Ket{ABC\cdots }
=C'(A) C'(B) C'(C)\cdots \Ket{\bar{A} \bar{B} \bar{C} \cdots }
$$

#### C变换的性质

$$
C^2 = 1
$$

$$
C^2\ket{A}
=C C'(A)\Ket{\bar{A}}
=C'\left(\bar{A} \right) C'\left(A \right) \ket{A}
$$

$$
C'(A) C'\left(\bar{A} \right)
=1,\quad
C'\left(\bar{A} \right)
=C'^*\left(A \right)
$$

若 $Q $ 是一个相加性守恒量，$\ket{A} $ 是 $Q $ 的任意本征态，满足

$$
Q \Ket{A } = Q'(A) \Ket{A },\quad
Q \Ket{\bar{A} } = -Q'(A) \Ket{\bar{A} }
$$

则

$$
Q C\ket{A}
=Q C'(A)\Ket{\bar{A}}
=-Q'(A) C'(A) \Ket{\bar{A} }
=-Q'(A) C \Ket{A }
=-C Q\ket{A}
$$

即：

$$
\left\{Q , C \right\}
=0
$$

也就是说，所有相加性守恒量都与C变换反对易。

### 纯中性态和C宇称

纯中性粒子在C变换下不变，其C变换因子称为C宇称。纯中性粒子才有C宇称。或者说，C宇称就是纯中性粒子C变换的本征值。

纯中性粒子C宇称的值为 $+1 $ 或 $-1 $

C宇称是相乘性守恒量，用 $C'=\pm $ 来代替 $C'=\pm 1 $

光子是纯中性粒子，其C宇称为 $-1 $，即 $C'(\gamma) = -1$

$\pi^0 $ 介子也是纯中性粒子，根据电磁相互作用C宇称守恒要求以及反应方程 $\pi^0 \to \gamma \gamma $，可知 $C'\left(\pi^0 \right)=C'(\gamma) C'(\gamma)=+1$

### 正反粒子组成系统的C宇称

只有在C变换下系统的组成不变，该系统才是纯中性系统。

一对正反粒子组成的系统，若正反粒子的自旋之和为 $S $，正反粒子之间的相对轨道角动量为 $L $，则这个纯中性系统的C宇称为：

$$
C'
=\left(-1 \right)^{L+S}
$$

### C变换和C宇称守恒

强相互作用和电磁相互作用在C变换下不变：通过C变换联系的两个过程规律和行为相同。

例子：$K^{*+}\to K^+ \pi^- $，C变换下 $K^{*-}\to K^-\pi^+ $，$K^{*\pm} $ 具有相同的质量、衰变宽度，相应衰变道的分支比也一致。

若初态是C变换的本征态，初态具有确定的C宇称，则末态也是C宇称的本征态，有和初态相同的C宇称，即**C宇称守恒**。

例子：已知 $\pi^0 $ 的C宇称为 $+ $，光子 $\gamma $ 的C宇称为 $- $，则允许：$\pi^0\to \gamma \gamma $；禁戒：$\pi^0\to \gamma \gamma \gamma $

例子：电子偶素是由正负电子构成的束缚态系统，是纯中性系统。这个纯中性系统由一对正反粒子组成，因此其C宇称为 $C'=(-1)^{L+S},L=0 , S=0,1$

$L=0,S=0,C'=\left(-1 \right)^{0}=+ $，允许 $e^+ e^-\to \gamma \gamma $，禁戒 $e^+ e^-\to \gamma \gamma \gamma  $

$L=0,S=1,C'=\left(-1 \right)^{1}=- $，禁戒 $e^+ e^-\to \gamma \gamma $，允许 $e^+ e^-\to \gamma \gamma \gamma  $

## 七、G变换

普通介子：除了同位旋对称性所包含的相加性守恒量 $I_3 $ 以及与 $I_3 $ 相关的电荷 $Q $ 之外，其他内部相加性守恒量都为 $0 $ 的粒子。

根据 $\displaystyle{Q=I_3 + \frac{b+S+C+B+T }{2 }  }$，由于在强相互作用中 $b,S,C,B,T $ 都是相加性守恒量，因此由普通介子的定义可知，普通介子满足 $Q=I_3 .$


G变换：一个系统绕同位旋空间第二轴 $I_2 $ 旋转 $\pi $ 后，再进行C变换

$$
G
=C R_2
=C \mathrm{e}^{\mathrm{i}\pi I_2}
$$

G变换就是除了 $I_3 $ 外，其他所有内部相加性守恒量都变号。因此，G变换的本征态必定是所有内部相加性守恒量都为零的粒子，即普通介子。

G宇称：普通介子除了 $I_3 $ 外，其他内部相加性守恒量为 $0 $，是G变换的本征态，其本征值称为G宇称。

普通介子的G宇称：$G'=(-1)^I C' $，其中 $C' $ 是普通介子C变换的系数，不一定是C宇称（对于纯中性普通介子，$C' $ 才是C宇称）。

轻子没有G宇称。$\pi^+ $ 介子有G宇称。质子和中子没有G宇称。$K^+ $ 介子有G宇称。

### G变换的性质

一切强子都有确定的G变换性质，但只有普通介子才有G宇称。

一切强子都有确定的C变换性质，但只有中性普通介子才有C宇称。

一个由多个强子构成的系统，只要其所具有的内部相加性守恒量除 $I_3 $ 和 $Q $ 外都为 $0 $，又有确定的同位旋，且 $I_3=0 $ 分量具有确定的C宇称，则这个态具有确定的G宇称。$G'=(-1)^{I} C' $

由一对正反粒子组成的具有确定轨道角动量 $L $ 和 $S $ 的系统：$G'=(-1)^{I+L+S} $

例子：两个正反 $K $ 介子组成的系统。$K $ 介子同位旋 $I=1/2 $，系统总同位旋可取 $I=0,1 $；$K $ 介子自旋 $S=0 $，系统总自旋 $S=0 $；当系统总同位旋取 $I=0 $，有 $G'=(-1)^{I+L+S}=(-1)^{L} $；当系统总同位旋取 $I=1 $，有 $G'=(-1)^{I+L+S}=(-1)^{L+1} $

由几个确定G宇称的子系统所组成的系统也具有确定的G宇称，其值等于各个子系统G宇称的乘积，即G宇称是相乘性守恒量。

例子：$\pi $ 介子G宇称为 $-1 $，$n $ 个 $\pi $ 介子构成的系统 G宇称为 $(-1)^n .$

## 八、P变换

P变换也称为空间反射变换。

$$
\vec{x} \to -\vec{x},\quad
t \to t
$$

P变换是分立变换。P变换下

$$
\vec{p} \to -\vec{p},\quad
E \to E
$$

$$
\vec{L}\equiv \vec{x} \times \vec{p}
\to \left(-\vec{x} \right) \times \left(-\vec{p} \right)
=\vec{L}
$$

$$
P \vec{L} = \vec{L} P
$$

$P,\vec{L} $ 有共同本征态，可同时具有确定测量值。

### 轨道宇称和内禀宇称

P变换满足 $P^2 = 1$，因此 $P $ 算符本征值可取 $\pm 1 $，$P $ 的本征值称为P宇称。

$P $ 的本征值为 $\pm 1 $，$P $ 的本征值称为 $P $ 宇称。

在P变换下

$$
\theta \to \pi-\theta,\quad
\phi \to \pi + \phi
$$

$$
P Y_{lm}(\theta,\phi)
=(-1)^L Y_{lm}(\theta,\phi)
$$

轨道宇称（与轨道角动量有关）：

$$
P' = (-1)^L
$$

P变换下粒子内部波函数有一定P宇称，称为内禀宇称，简称粒子的宇称。

### 相对宇称和绝对宇称

宇称具有相对性，即内禀宇称只能根据 P 宇称守恒定律的要求来确定。只有纯中性粒子才具有绝对内禀宇称，如：

$$
P'(\gamma) = -1,\quad
P'\left(\pi^0 \right) = -1
$$


一对正反粒子组成的纯中性系统，系统的轨道角动量为 $L $，总自旋为 $S $，系统的态可用 $\ket{x,r,-x,t} $ 描述。$r,t $ 分别描述两个粒子的自旋状态。对这个态作 CP 联合变换：

$$
CP\ket{x,r,-x,t}
=C\ket{-x,r,x,t}
=\Ket{x,t,-x,r }
$$

交换自旋可以进一步给出：

$$
=\left(-1 \right)^{S+1} \Ket{x,r,-x,t },\quad \mathrm{费米子}
$$

$$
=\left(-1 \right)^{S} \Ket{x,r,-x,t },\quad \mathrm{玻色子}
$$

已知系统C宇称为 $(-1)^{L+S} $，可以得到系统绝对宇称为

$$
P'=(-1)^{L+1},\quad \mathrm{费米子}
$$

$$
P'=(-1)^{L},\quad \mathrm{玻色子}
$$

而轨道宇称的贡献为 $(-1)^L $，总之，一对正反粒子组成系统的**内禀宇称**为：$P'=-1 $，正反费米子对；$P'=+1 $，正反玻色子对。

对于非纯中性粒子的**内禀宇称**，作约定：

**电荷 $Q$**：同一同位旋多重态的不同电荷态宇称相同  

**量子数**：$P'(N) = +1$  

**$e$ 轻子数 $L_e$**：$P'(e^-) = +1$  

**$\mu$ 轻子数 $L_\mu$**：$P'(\mu^-) = +1$  

**$\tau$ 轻子数 $L_\tau$**：$P'(\tau^-) = +1$  

**奇异数 $S$**：$P'(K) = P'(\pi) = +1$  

**粲数 $C$**：$P'(D) = P'(\pi) = +1$  

**底数 $B$**：$P'(B) = P'(\pi) = +1$  

规定核子的宇称为 $+1$，根据正反费米子所构成系统的内禀宇称为 $-1$，可以确定反核子的宇称为 $-1$，因此，正反费米子的宇称正好相反。

### 宇称守恒和宇称不守恒

P宇称在强相互作用和电磁相互作用过程中守恒，在弱相互作用过程中不守恒。

对于不直接参与强相互作用和电磁相互作用的中微子，其没有确定的 P 宇称。

## CP变换

### CPT定理

若讨论的场是定域场，即场对应的粒子是点粒子；且场具有相对论所要求的正 Lorentz 协变性；且满足自旋统计关系：自旋为整数的粒子满足玻色统计，自旋为半整数的粒子满足费米统计；则运动规律在 CPT 联合变换下不变。

CPT 联合变换下不变表现为：粒子与反粒子的质量、寿命、自旋磁矩的 $g $ 因子都完全相同；粒子的某一衰变速率和 C 变换后反粒子相应衰变道的衰变速率相同。

### 弱相互作用的CP不变性

弱相互作用中宇称可以不守恒。弱相互作用中 C 宇称也可以不守恒。

但在弱相互作用具有 CP 不变性，即弱相互作用过程中 CP 宇称守恒。

物理含义：两个通过 CP 变换相联系的过程，其演化性质和概率分布完全相同。

### CP破坏现象

弱相互作用具有 CP 不变性，则 $K_S $ 和 $K_L $ 都应该是 CP 变换的本征态。$K_S $ 的 CP 宇称为 $+1 $，$K_L $ 的 CP 宇称为 $-1 $

$K_S $ 可以衰变到两个 $\pi $ 介子，而 $K_L $ 不能衰变到两个 $\pi $ 介子。

但 1964 年在实验中发现 $K_L $ 有一小部分可衰变到两个 $\pi $ 介子，也即弱相互作用过程中存在 CP 破坏的分量。

定义：

$$
\left|\eta_{+-} \right|
\equiv \left[\frac{\Gamma\left(K_L\to \pi^+ \pi^- \right) }{\Gamma\left(K_S\to \pi^+ \pi^- \right) }  \right]^{1/2},\quad
\left|\eta_{00} \right|
\equiv \left[\frac{\Gamma\left(K_L\to \pi^0 \pi^0 \right) }{\Gamma\left(K_S\to \pi^0 \pi^0 \right) }  \right]^{1/2}
$$

若弱相互作用具有 CP 变换不变性，则 $\left|\eta_{+-} \right|=\left|\eta_{00} \right|=0 $

然而实验结果表明，$\left|\eta_{+-} \right|=2.286\times 10^{-3},\quad \left|\eta_{00} \right|=2.274\times 10^{-3} $，即在若相互作用下，CP 破坏效应约占千分之二。

此外，CP 变换还体现在 $K_L $ 粒子的半轻子衰变的测量。

## 十、全同粒子交换变换

$P_{ij} $ 交换第 $i $ 个粒子与第 $j $ 个粒子。

$$
P_{ij}^2 = 1
$$

$$
P_{ij} H = H P_{ij}
$$

$P_{ij} $ 在各种相互作用下是守恒量。

$P_{ij} $ 的本征值为 $\pm 1 $，它把一切粒子分为玻色子和费米子。

$P_{ij} $ 是严格守恒量，而 C 变换和 P 变换在弱相互作用下不守恒。

### 两个全同粒子组成的系统的选择规则

两粒子之间的轨道角动量记为 $L $，总自旋记为 $S $，$P_{ij} $ 作用后，费米子：$\left(-1 \right)^{L+S+1}=-1 $；玻色子：$\left(-1 \right)^{L+S}=+1 $

无论费米子还是玻色子，两个全同粒子组成的系统满足：

$$
L+S = 偶数
$$

### 广义全同粒子组成系统的选择规则

同位旋对交换算符本征值的贡献：对半整数同位旋粒子贡献 $\left(-1 \right)^{I+1} $，对整数同位旋粒子贡献 $\left(-1 \right)^I $，$I $ 为两个粒子的总同位旋。

考虑同位旋的全同性后的选择定则：

$$
L+S+I+2i = 偶数
$$

举例：

---

氘核：由质子和中子构成，质子和中子可以作为全同粒子。它们的自旋和同位旋都是 $1/2 $，满足

$$
L+S+I+2i = 偶数
$$

实验上确定 $S=1 $，$L $ 是 $0 $ 和 $2 $ 的混合，因此 $I $ 必须是偶数。

而中子和质子构成的系统的总同位旋 $I=0 $ 或 $I=1 $，因此只能是 $I=0 .$

这就说明，没有与氘核对应的 $(pp) $ 或 $(nn) $ 束缚态存在。

$I=0,(pn):I=0,I_3=0 $；

$I=1,(pn):I=1,I_3=0;(pp):I=1,I_3=+1;(nn):I=1,I_3=-1 $

## 一、弱相互作用现象

弱相互作用是自然界四种基本相互作用之一，其特点是强度最弱（相对于强相互作用和电磁相互作用）且作用范围极短。它主要导致粒子的衰变，特别是那些寿命相对较长的衰变过程。

### 1. 弱衰变过程的特点

$\text{n} \rightarrow \text{p} + \text{e}^{-} + \overline{\nu}_{\text{e}},\quad \mu^{-} \rightarrow \text{e}^{-} + \overline{\nu}_{\text{e}} + \nu_{\mu},\quad \pi^{-} \rightarrow \mu^{-} + \overline{\nu}_{\mu},\quad \Lambda \rightarrow \text{p} + \pi^{-},\quad \text{K}^{-} \rightarrow \mu^{-} + \overline{\nu}_{\mu} $

弱相互作用的特点：(a) 寿命长：表明相互作用的强度很弱！；(b) 宇称不守恒 

### 2. 普适费米弱相互作用理论

发展历程：
    早期费米理论 (Fermi's Theory, 1934)：
        恩里科·费米基于电磁相互作用的经验，提出了一个描述 $\beta$ 衰变的理论，假设衰变发生在空间中的一个“点”上，即所有参与粒子在同一点相互作用。
        相互作用拉格朗日量（或哈密顿量密度）形式：$\mathcal{L}_{\text{int}} = G_F (\overline{\psi}_{\text{p}} \gamma^\mu \psi_{\text{n}})(\overline{\psi}_{\text{e}} \gamma_\mu \psi_{\nu_{\text{e}}})$
            其中 $\psi$ 代表粒子的狄拉克场，$\gamma^\mu$ 是狄拉克矩阵。
            $G_F$ 是费米耦合常数，其数值约为 $1.166 \times 10^{-5} \text{ GeV}^{-2}$，量纲是能量的倒数平方。这个小数值解释了弱相互作用的弱强度。

    宇称不守恒后的修正 (V-A 理论)：
        在宇称不守恒被实验证实后，费米理论需要修正以纳入这一新发现。
        矢量-轴矢量 (V-A) 理论：由乔治·盖尔曼 (Murray Gell-Mann) 和理查德·费曼 (Richard Feynman) 以及恩纳卡尔·苏达山 (E.C.G. Sudarshan) 和罗伯特·马沙克 (Robert Marshak) 提出。
        实验表明，弱相互作用流具有纯粹的 V-A 形式。这意味着弱相互作用只耦合粒子的左手手征分量（或反粒子的右手手征分量）。
        弱流的形式：$J^\mu = \overline{\psi}_1 \gamma^\mu (1 - \gamma^5) \psi_2$
            这里的 $(1 - \gamma^5)$ 是一个手征投影算符，它从 $\psi_2$ 中投影出左手分量，并与 $\overline{\psi}_1$ 的左手分量耦合。
            这直接解释了为什么中微子总是左螺旋的，因为它们只与弱相互作用耦合，而弱相互作用只作用于左手粒子。

普适性 (Universality) 与夸克混合：
    弱相互作用对所有轻子和夸克族（代）中的粒子都以几乎相同的基本强度发生。这被称为弱相互作用的普适性。
    卡比博角 (Cabibbo Angle, $\theta_C$, 1963)：
        实验发现，涉及奇异夸克 ($\text{s}$) 的弱衰变（如 $\text{s} \rightarrow \text{u}$）比不涉及奇异夸克（如 $\text{d} \rightarrow \text{u}$）的弱衰变强度略小。
        尼古拉·卡比博 (Nicola Cabibbo) 提出，这是因为在弱相互作用中，下型夸克（$\text{d}, \text{s}$）并不是纯粹的本征态，而是以一种混合态参与弱相互作用。
        他引入了一个混合角 $\theta_C$，使得弱相互作用中的夸克本征态是原始夸克本征态的线性组合：
            $$ \begin{pmatrix} d' \\ s' \end{pmatrix} = \begin{pmatrix} \cos\theta_C & \sin\theta_C \\ -\sin\theta_C & \cos\theta_C \end{pmatrix} \begin{pmatrix} d \\ s \end{pmatrix} $$
            其中，$d'$ 和 $s'$ 是参与弱相互作用的“混合夸克”。
        因此，$\text{d} \leftrightarrow \text{u}$ 跃迁的强度与 $\cos^2\theta_C$ 成正比，而 $\text{s} \leftrightarrow \text{u}$ 跃迁的强度与 $\sin^2\theta_C$ 成正比。这成功解释了不同夸克衰变率的差异。
    GIM 机制 (Glashow-Iliopoulos-Maiani Mechanism, 1970)：
        在只有三夸克（u, d, s）的理论中，卡比博理论预言了某些“味无关中性流 (Flavor-Changing Neutral Currents, FCNC)”过程（例如 $\text{K}^0 \rightarrow \mu^+ \mu^-$），但实验上这些过程的发生率被极大地压制。
        谢尔登·格拉肖 (Sheldon Glashow)、扬尼斯·伊利奥普洛斯 (John Iliopoulos) 和卢西奥·马亚尼 (Luciano Maiani) 提出，如果存在第四种夸克——粲夸克 ($\text{c}$)，那么通过粲夸克的引入，可以使得所有预言的 FCNC 过程通过一个“GIM 抵消”机制相互抵消，从而解释了实验上观察到的极低发生率。这一机制的成功预言了粲夸克的存在，其在 1974 年 J/$\psi$ 粒子的发现中得到证实。
    CKM 矩阵 (Cabibbo-Kobayashi-Maskawa Matrix, 1973)：
        当第三代夸克（顶夸克 $\text{t}$ 和底夸克 $\text{b}$）被发现后，卡比博理论被推广到三代夸克。
        小林诚 (Makoto Kobayashi) 和益川敏英 (Toshihide Maskawa) 提出，三代夸克之间的弱相互作用混合可以用一个 $3 \times 3$ 的酉矩阵来描述，这就是 CKM 矩阵：
            $$ V_{CKM} = \begin{pmatrix} V_{ud} & V_{us} & V_{ub} \\ V_{cd} & V_{cs} & V_{cb} \\ V_{td} & V_{ts} & V_{tb} \end{pmatrix} $$
            其中每个元素 $V_{ij}$ 代表夸克 $\text{i}$ 和 $\text{j}$ 之间通过弱相互作用发生跃迁的耦合强度。
        CKM 矩阵的关键意义在于，当夸克家族数大于等于三时，该矩阵的复相位可以自然地引入 CP 破坏 (CP Violation) 现象。CP 破坏是解释宇宙中物质-反物质不对称（即为什么我们周围都是物质而不是反物质）的关键机制之一。

### 3. 中间玻色子理论和电弱统一的可能性

费米理论的局限性：
    费米理论是一个点式相互作用理论，这导致它在计算高能散射过程时会出现发散问题，即不可重整化。
    费米耦合常数 $G_F$ 的量纲是 $[\text{G}_F] = \text{能量}^{-2}$。这意味着在能量尺度达到 $G_F^{-1/2}$ 附近（约 300 GeV）时，理论的预言能力失效，暗示了在更高能量下弱相互作用的性质会发生变化。

中间玻色子理论：
    为了克服费米理论的局限性，并使其成为一个可重整化的理论，物理学家们提出弱相互作用是通过交换带质量的矢量玻色子（Intermediate Vector Bosons）来传递的。
    预言的弱相互作用传递子：
        带电 $W$ 玻色子 ($W^+, W^-$)：负责传递带电弱相互作用，例如 $\beta$ 衰变。它们带有单位电荷。
        中性 $Z$ 玻色子 ($Z^0$)：负责传递中性弱相互作用，不涉及电荷转移或味改变。
    相互作用范围与玻色子质量：相互作用的范围与传递玻色子的质量成反比 ($R \approx \hbar / (Mc)$)。由于弱相互作用的作用范围极短（约 $10^{-18}$ 米），这强烈预示着 $W$ 和 $Z$ 玻色子必须具有非常大的质量。
        费米耦合常数 $G_F$ 可以与 $W$ 玻色子的质量 $M_W$ 关联起来：$G_F \approx \frac{g^2}{M_W^2}$，其中 $g$ 是弱耦合常数。
    可重整化性：通过交换规范玻色子来描述相互作用，可以构建一个规范理论 (Gauge Theory)。规范理论通常是可重整化的，这意味着可以通过系统性的重整化程序去除计算中的无限大项，得到有限且有物理意义的预测。

电弱统一的可能性：
    在 1960 年代，史蒂文·温伯格 (Steven Weinberg)、谢尔登·格拉肖 (Sheldon Glashow) 和阿卜杜斯·萨拉姆 (Abdus Salam) 独立地提出，电磁相互作用和弱相互作用并非完全独立的，而是可以被统一在一个更宏大的理论框架下——电弱统一理论 (Electroweak Unification Theory)。
    核心思想：这两种相互作用都源于一个更大的对称群的自发破缺 (Spontaneous Symmetry Breaking)。这意味着在更高的能量或更基本的层面上，它们是统一的；而在我们通常观察到的低能世界，由于对称性破缺，它们表现出不同的强度和特征（例如，光子无质量，而 $W/Z$ 玻色子有巨大质量）。

## 四、电弱统一理论 (标准模型的核心部分)

电弱统一理论是粒子物理标准模型的重要基石，它成功地将电磁相互作用和弱相互作用描述为一个统一的规范理论。

规范对称性 (Gauge Symmetry)：
    电弱理论的基础是 $SU(2)_L \times U(1)_Y$ 规范群。这是一个非阿贝尔群 ($SU(2)_L$) 和一个阿贝尔群 ($U(1)_Y$) 的乘积。
        $SU(2)_L$ 规范对称性：与弱同位旋 (Weak Isospin) 相关联。这个对称性仅作用于粒子的左手手征分量 (Left-handed chiral components)。
            轻子和夸克被组织成弱同位旋双态（二元组）：
                轻子双态：$\begin{pmatrix} \nu_e \\ e^- \end{pmatrix}_L$, $\begin{pmatrix} \nu_\mu \\ \mu^- \end{pmatrix}_L$, $\begin{pmatrix} \nu_\tau \\ \tau^- \end{pmatrix}_L$
                夸克双态：$\begin{pmatrix} u \\ d' \end{pmatrix}_L$, $\begin{pmatrix} c \\ s' \end{pmatrix}_L$, $\begin{pmatrix} t \\ b' \end{pmatrix}_L$ （其中 $d', s', b'$ 是通过 CKM 矩阵混合后的下型夸克）
            这种对称性要求存在 3 个无质量的规范玻色子：$W^1, W^2, W^3$。
        $U(1)_Y$ 规范对称性：与弱超荷 (Weak Hypercharge) 相关联。
            这个对称性要求存在 1 个无质量的规范玻色子：$B$。
    在规范对称性未破缺的阶段，这 4 个规范玻色子都必须是无质量的。然而，为了与实验观察到的 $W, Z$ 玻色子的大质量相符，我们需要引入自发对称破缺。

自发对称破缺 (Spontaneous Symmetry Breaking) 和希格斯机制 (Higgs Mechanism)：
    目的：为了在保持规范理论可重整化性的同时，赋予规范玻色子（以及费米子）质量。
    希格斯场 (Higgs Field)：引入一个特殊的复标量场，具有非零的自旋（标量，自旋为 0），其在真空中具有非零的期望值 (Vacuum Expectation Value, VEV)。
        这个非零的 VEV 意味着即使在最低能量状态（真空）下，宇宙空间也“充满了”希格斯场，就像一个无处不在的介质。
    机制：
        当 $W^1, W^2, W^3, B$ 这四个无质量的规范玻色子穿过这个充满希格斯场的真空时，它们会与希格斯场发生相互作用。这种相互作用赋予了它们“有效质量”。
        经过希格斯机制后，原始的规范玻色子会混合并获得质量：
            $W^+$ 和 $W^-$ 玻色子：$W^1$ 和 $W^2$ 混合形成带电的 $W^+$ 和 $W^-$ 玻色子：
                $$W^\pm = \frac{1}{\sqrt{2}}(W^1 \mp i W^2)$$
                它们获得了质量 $M_W \approx 80.4 \text{ GeV}/\text{c}^2$。
            光子 ($\gamma$) 和 $Z^0$ 玻色子：$W^3$ 和 $B$ 混合形成两种物理粒子：
                光子 ($\gamma$)：作为电磁相互作用的传递者，它保持无质量。
                $Z^0$ 玻色子：作为中性弱相互作用的传递者，它获得了质量 $M_Z \approx 91.2 \text{ GeV}/\text{c}^2$。
                $$A^\mu = B^\mu \cos\theta_W + W_3^\mu \sin\theta_W \quad (\text{光子})$$
                $$Z^\mu = -B^\mu \sin\theta_W + W_3^\mu \cos\theta_W \quad (Z^0 \text{玻色子})$$
                其中 $\theta_W$ 是弱混合角 (Weak Mixing Angle) 或 温伯格角 (Weinberg Angle)。它通过 $\sin^2\theta_W \approx 0.231$ 描述了电磁和弱相互作用之间的联系。
            质量关系：希格斯机制预言了 $W$ 和 $Z$ 玻色子质量之间的精确关系：$M_Z = \frac{M_W}{\cos\theta_W}$。这一关系在实验上得到了高度精确的验证。
    预言希格斯玻色子 (Higgs Boson)：
        希格斯机制不仅赋予了规范玻色子质量，还会留下一个额外的、有质量的、自旋为零的标量粒子——即希格斯玻色子。
        希格斯玻色子是希格斯场的量子激发。它的发现（2012 年在欧洲核子研究组织 CERN 的大型强子对撞机 LHC）是粒子物理标准模型最重要的里程碑之一，因为它直接证实了粒子获得质量的机制。

电弱统一理论中包含的基本粒子：
    费米子 (Fermions)：构成物质的基本粒子，具有半整数自旋 ($1/2 \hbar$)。
        轻子 (Leptons)：不感受强相互作用。分为三代，每代包含一个带电轻子和一个对应中微子。
            第一代：电子 ($e^-$), 电子中微子 ($\nu_e$)
            第二代：渺子 ($\mu^-$), 渺中微子 ($\nu_\mu$)
            第三代：陶子 ($\tau^-$), 陶中微子 ($\nu_\tau$)
            每个轻子都有对应的反粒子。
        夸克 (Quarks)：感受强相互作用，并通过强相互作用结合成强子（如质子、中子）。分为三代。
            第一代：上夸克 ($u$), 下夸克 ($d$)
            第二代：粲夸克 ($c$), 奇异夸克 ($s$)
            第三代：顶夸克 ($t$), 底夸克 ($b$)
            每个夸克都有对应的反夸克，并且每种夸克还有三种“颜色”（红、绿、蓝），这是强相互作用的荷。
        手征性：所有费米子都有左手和右手分量。但在电弱相互作用中，只有左手费米子（和右手反费米子）参与弱相互作用。右手费米子不直接参与弱相互作用（除非它们通过与希格斯场的耦合获得质量）。

    规范玻色子 (Gauge Bosons)：传递基本相互作用的粒子，具有整数自旋 ($1 \hbar$)。
        光子 ($\gamma$)：电磁相互作用的传递者。无质量。
        $W^+, W^-$ 玻色子：带电弱相互作用的传递者。带单位电荷，质量巨大。它们介导了改变夸克和轻子“味”的衰变（如 $\beta$ 衰变）。
        $Z^0$ 玻色子：中性弱相互作用的传递者。不带电荷，质量巨大。它介导了不改变粒子“味”的弱相互作用（如中微子散射）。
        胶子 ($g$)：强相互作用的传递者（不属于电弱理论，但包含在标准模型中）。无质量。有 8 种不同的胶子，它们带有“色荷”并介导夸克之间的强力。

    希格斯玻色子 (Higgs Boson)：
        一个标量粒子（自旋为 0），是希格斯场的量子激发。
        它通过与费米子和 $W/Z$ 玻色子的汤川耦合 (Yukawa Coupling) 赋予它们质量。粒子与希格斯场的耦合越强，其获得的质量越大。

粒子之间的相互作用机理：
    电磁相互作用：
        传递子：光子 ($\gamma$)。
        作用对象：所有带有电荷的粒子（夸克、带电轻子如 $e^-, \mu^-, \tau^-$）。
        特征：长程作用力，强度相对较强。描述了原子和分子的形成，光与物质的相互作用。
    带电流弱相互作用 (Charged-Current Weak Interaction)：
        传递子：$W^+$ 或 $W^-$ 玻色子。
        作用对象：左手费米子，并且在相互作用过程中会改变粒子的味 (Flavor)。
        典型过程：夸克之间的转化（例如 $d \rightarrow u$，涉及 $W^-$ 辐射），轻子之间的转化（例如 $e \leftrightarrow \nu_e$）。这是引起大多数粒子衰变的原因，如中子 $\beta$ 衰变、渺子衰变等。
    中性流弱相互作用 (Neutral-Current Weak Interaction)：
        传递子：$Z^0$ 玻色子。
        作用对象：左手费米子，但不改变粒子的味。
        典型过程：不涉及电荷转移或味改变的弱相互作用。最典型的例子是中微子与物质的弹性散射（例如 $\nu_\mu + e^- \rightarrow \nu_\mu + e^-$），在这种情况下，中微子的味没有改变。中性流的发现是电弱统一理论的重要实验证据。

未来展望：
    高亮度大型强子对撞机 (High-Luminosity LHC, HL-LHC) 项目：
        这是 CERN LHC 的重大升级计划，旨在大幅提高对撞机的亮度（单位时间内产生的对撞次数），从而积累更多的数据样本。
        目标是更精确地测量标准模型的参数，特别是希格斯玻色子的性质，并寻找超越标准模型的新物理迹象，例如稀有衰变或暗物质粒子。
        HL-LHC 的成功完成将是欧洲粒子物理的焦点。
    未来对撞机计划 (Future Colliders)：粒子物理界正在积极规划下一代对撞机，以探索更高的能量尺度和更深层次的物理规律。
        希格斯工厂 (Higgs Factory)：例如国际直线对撞机 (International Linear Collider, ILC) 或未来环形对撞机 (Future Circular Collider electron-positron, FCC-ee)。这些对撞机旨在通过产生大量希格斯玻色子，对其性质进行超高精度的测量，以揭示其与标准模型预言的偏差，从而探寻新物理。
        更高能量的强子对撞机：例如未来环形对撞机 (Future Circular Collider hadron-hadron, FCC-hh)。这些对撞机将达到比 LHC 高一个量级的对撞能量，以直接探测更高质量的新粒子，解决标准模型无法解释的重大问题，如暗物质、中微子质量的起源、引力的量子化以及物质-反物质不对称的更深层原因。
    这些雄心勃勃的科学目标将推动粒子物理领域乃至整个科学界的技术和科学探索进入新的未知领域，并对基础科学和社会发展产生深远影响。
